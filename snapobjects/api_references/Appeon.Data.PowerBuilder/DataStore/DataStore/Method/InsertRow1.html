Remarks

InsertRow只是插入行而不更改显示或设置当前行。要滚动到该行并使其成为当前行，请调用ScrollToRow。要简单地将其设为当前行，请调用SetRow。 新插入的行（状态标志为New）不包括在修改的计数中，直到在行中输入数据（其状态标志变为NewModified）。
