Remarks

SetItem在DataStore缓冲区中设置一个值。它不会影响当前行和列中编辑控件中当前的值，即当前用户已更改或可能更改的数据。在验证和接受之前，编辑控件中的值不会成为DataStore项的值。在脚本中，您可以使用SetText Method更改编辑控件中的值。 如果要在DataStore控件或将脚本作为源的DataStore中设置项的值，可以使用SetItem。
