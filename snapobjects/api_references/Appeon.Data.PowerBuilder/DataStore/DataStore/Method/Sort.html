Remarks

Sort使用DataStore的当前排序条件。要更改排序条件，请使用SetSort Method。 SetSort Method相当于在DataStore画家的Rows菜单上使用Sort命令。如果在调用Sort之前未调用SetSort设置排序条件，则Sort将使用DataStore对象定义中指定的排序条件。当Retrieve Method检索DataStore的数据时，PowerBuilder会应用为DataStore对象定义的排序条件（如果有）。只有在使用SetSort更改排序条件或者由于处理或用户输入而导致数据发生更改后，才需要调用Sort。

在指定列上对DataStore进行排序时，无论您是为排序条件选择升序还是降序，具有空数据的行都保留在顶部。排序顺序是对从数据库返回的结果集执行的，但在SQL查询中使用ORDER BY子句时，不一定是数据库使用的相同排序顺序（返回结果集）。

Sort Method使用典型的词法排序，其中符号（如连字符或下划线）的排名高于字母数字字符。它以与字典相同的方式比较字符。

如果设置了“需要检索”选项，则Sort Method会取消其效果。排序会导致检索所有行，以便正确排序。它还将当前行更改为1，而不会触发RowFocusChanged或RowFocusChanging Event。调用Sort Method后，应以编程方式触发这些Event。
