IDataStore.RetrieveStart(object sender, DwRetrieveEventArgs e) Method

.NET Standard 2.x | Current Version(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

This method is called when the DataStore is about to begin the data retrieval. It triggers the OnRetrieveStart event by default, and it can be overridden in a child class of the DataStore.

Namespace: Appeon.Data.PowerBuilder

Assembly: Appeon.Data.PowerBuilder.dll

Syntax

  public void RetrieveStart(object sender, DwRetrieveEventArgs e);

Parameters

sender System.Object

An event sender in which the event occurred.

It is a reference to the current DataStore by default.

e Appeon.Data.PowerBuilder.DwRowChangeEventArgs

A DwRetrieveEventArgs object which can be used to pass by event arguments.

Remarks

In the OnRetrieveStart event: To prevent previously retrieved data from being cleared, set DwRetrieveEventArgs.IsResetBuffer to false to allow the current retrieval process to append new rows to the old data; to cancel the retrieval, set DwRetrieveEventArgs.IsCancelto true.

Examples

The following code example gets the RowCount value when Retrieve is about to begin and when it is completed.

using Appeon.Data.PowerBuilder;
using System;

namespace Appeon.ApiDoc.IDataStoreExamples
{
    public class RetrieveStartExample
    {
        private SchoolContext _context;

        public RetrieveStartExample(SchoolContext dataContext)
        {
            // Sets the data context
            _context = dataContext;
        }

        public void Example()
        {
            // Instantiates datastore with datawindow: d_department
            var datastore = new DataStore("d_department", _context);

            // Defines what to do when Retrieve is about to begin
            datastore.OnRetrieveStart +=
                (sender, args) =>
                {
                    // Executes the following code before Retrieve is executed 
                    Console.WriteLine("Before Retrieve, Rowcount: {0}",
                        args.RowCount);
                };

            // Retrieves rows from the database for datastore
            datastore.Retrieve();

            Console.WriteLine("After Retrieve, Rowcount: {0}", datastore.RowCount);

            /*This code produces the following output:
            
            Before Retrieve, Rowcount: 0
            After Retrieve, Rowcount: 4           
            */

        }
    }
}
