获取是否可以就地更新键列，或者是否必须删除行并重新插入。

Gets whether the key column can be updated in place or the row has to be deleted and reinserted.

命名空间： Appeon.Data.PowerBuilder

程序集： Appeon.Data.PowerBuilder.dll

语法

   bool UpdateKeyInPlace { get; set; }

Property Value

System.Boolean

This value determines the syntax DataStore generates when the value of a key column is modified:

Yes -- Use the UPDATE statement when the key is changed so that the key is updated in place.

No -- Use a DELETE and an INSERT statement when the key is changed.

using Appeon.Data.PowerBuilder;
using System;

namespace Appeon.ApiDoc.IDwTableExamples
{
    public class UpdateKeyInPlaceExample
    {
        private readonly SchoolContext _context;

        public UpdateKeyInPlaceExample(SchoolContext context)
        {
            _context = context;
        }

        public void Example()
        {
            // 创建一个datastore对象
            IDataStore course = new DataStore("d_course", _context);

            // 获取该datastore对象使用的table
            IDwTable table = course.Object.DataWindow.Table;

            // 获取DataStore中的
            var express = table.UpdateKeyInPlace;

            // 输出
            Console.WriteLine("Update Key = {0}", express.ToString());

            /*This code produces the following output:
            
            Update Key = False
            */
        }
    }
}
