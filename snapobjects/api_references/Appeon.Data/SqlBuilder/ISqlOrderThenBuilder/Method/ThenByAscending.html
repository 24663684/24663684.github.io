ISqlOrderThenBuilder .ThenByAscending(params string[] orderExpressions) 方法

.NET Standard 2.x | 当前版本(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

在ORDER BY clause上添加一个或者多个升序排序条件。

Adds one or more ascending sort criteria to the ORDER BY clause.

命名空间： Appeon.Data

程序集： Appeon.Data.dll

语法

  ISqlOrderThenBuilder ThenByAscending(params string[] orderExpressions);

Parameters

orderExpressions System.String[]

(Optional) An array of expressions which specify a column or an expression on which to sort the query result set.

Returns

Appeon.Data.ISqlOrderThenBuilder

Returns an ISqlOrderThenBuilder object which can be used to add more sort criteria to the current ORDER BY clause.

Examples

演示ThenByAscending()方法。以下案例查询Course表中的记录，并按DepartmentID升序，然后按Title升序

using Appeon.Data;
using System;

namespace Appeon.ApiDoc.ISqlOrderThenBuilderExamples
{
    public class ThenByAscendingExample
    {
        private SchoolContext _context;

        public ThenByAscendingExample(SchoolContext dataContext)
        {
            // Set Data Context
            _context = dataContext;
        }

        public void Example()
        {
            // 声明SqlQueryBuilder
            var sqlbuilder = new SqlQueryBuilder();

            // 定义SQL语句，查询Course表中的所有记录，并按DepartmentID升序，然后按Title升序
            sqlbuilder.Select("*")
                .From("Course", "C")
                .OrderByAscending("DepartmentID")
                .ThenByAscending("Title");

            // 将sqlbuilder语法转换成context对应数据库的可执行语句
            string sql = sqlbuilder.ToSqlString(_context);

            Console.WriteLine(sql);
            
            /*This code produces the following output:
            
            SELECT
            *
            FROM [Course] AS [C]
            ORDER BY
            [DepartmentID] ASC,
            [Title] ASC
            */
        }
    }
}
