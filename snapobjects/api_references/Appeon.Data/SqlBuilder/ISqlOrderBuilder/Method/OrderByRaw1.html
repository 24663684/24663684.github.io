创建ORDER BY子句并用raw SQL指定一个或者多个排序条件。

Creates an ORDER BY clause and specifies one or more sort criteria with raw SQL.

命名空间： Appeon.Data

程序集： Appeon.Data.dll

语法

  ISqlOrderThenBuilder OrderByRaw(string orderClause);

Parameters

orderClause System.String

A string of raw SQL which specifies one or more sort criteria.

Returns

Appeon.Data.ISqlOrderThenBuilder

Returns an ISqlOrderThenBuilder object which can be used to add more sort criteria to the current ORDER BY clause.

Examples

演示OrderByRaw(string orderClause)方法。以下案例查询Department表所有记录，指定排序条件，按budget升序排列asc表示升序，desc表示降序

using Appeon.Data;
using Appeon.Data.PowerBuilder;
using System;

namespace Appeon.ApiDoc.ISqlOrderBuilderExamples
{
    public class OrderByRawExample
    {
        private SchoolContext _context;

        public OrderByRawExample(SchoolContext dataContext)
        {
            // Set Data Context
            _context = dataContext;
        }

        public void Example1()
        {
            // 声明SqlQueryBuilder
            var sqlbuilder = new SqlQueryBuilder();

            // 定义SQL语句，查询Department表所有记录，并按budget升序排列
            sqlbuilder.Select("*")
                .From("Department", "dept")
                .OrderByRaw("budget asc");

            // 将sqlbuilder语法转换成context对应数据库的可执行语句
            string sql = sqlbuilder.ToSqlString(_context);

            Console.WriteLine(sql);
            
            /*This code produces the following output:
            
            SELECT
            *
            FROM [Department] AS [dept]
            ORDER BY
            [budget] ASC
            */
        }
    }
}
