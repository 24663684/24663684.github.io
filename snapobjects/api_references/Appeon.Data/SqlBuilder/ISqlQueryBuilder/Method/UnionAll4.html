ISqlQueryBuilder.UnionAll(string sqlRaw, string builderAlias) 方法

.NET Standard 2.x | 当前版本(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

使用UNION ALL将当前SQL SELECT语句的结果集与由另一个raw SQL SELECT语句的结果集进行合并 (合并的结果集中包括重复记录)。并指定该raw SQL的别名。

Uses UNION ALL to combine the result set of the current SQL SELECT statement and the result set of another raw SQL SELECT statement (the combined result set includes duplicate records). And specifies an alias for the raw SQL SELECT statement.

命名空间： Appeon.Data

程序集： Appeon.Data.dll

语法

  ISqlQueryBuilder UnionAll(string sqlRaw, string builderAlias);

Parameters

sqlRaw System.String

The raw SQL SELECT whose result set is to be combined with the current SQL query.

builderAlias System.String

The alias of the raw SQL SELECT statement.

Returns

Appeon.Data.ISqlQueryBuilder

Returns the current ISqlQueryBuilder object.

Examples

演示UnionAll(string sqlRaw, string builderAlias)方法。用于合并两个或多个SELECT语句的所有结果集,包括重复行，不进行排序SELECT语句必须拥有相同数量的列, 列也必须拥有相似的数据类型,并且2个语句列顺序相同.并设置别名

using Appeon.Data;
using System;

namespace Appeon.ApiDoc.ISqlQueryBuilderExamples
{
    public class UnionAllExample
    {
        private SchoolContext _context;

        public UnionAllExample(SchoolContext dataContext)
        {
            // Set Data Context
            _context = dataContext;
        }

        public void Example4()
        {
            // 声明SqlQueryBuilder
            var sqlbuilder = new SqlQueryBuilder();

            sqlbuilder.Select("courseid", "courseid")
                .Select("url", "address")
                .From("OnlineCourse", "online");

            // 合并的Select语句是字符串，需要保证与sqlbuilder中列数，数据类型及顺序相同
            // 并设置合并语句的别名为test
            sqlbuilder.UnionAll("select courseid as courseid, location as address" 
                + " from OnsiteCourse", "test");

            // 将sqlbuilder语法转换成context对应数据库的可执行语句
            string sql = sqlbuilder.ToSqlString(_context);

            Console.WriteLine(sql);
            
            /*This code produces the following output:
            
            SELECT
            [courseid] AS [courseid],
            [url] AS [address]
            FROM [OnlineCourse] AS [online]
            UNION ALL
            (SELECT
            [courseid] AS [courseid],
            [location] AS [address]
            FROM [OnsiteCourse])
            */
        }
    }
}
