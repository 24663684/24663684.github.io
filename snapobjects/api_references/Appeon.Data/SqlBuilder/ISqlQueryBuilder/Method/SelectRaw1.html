ISqlQueryBuilder.SelectRaw(string selectRaw, params ISqlParameter[] parameters) 方法

.NET Standard 2.x | 当前版本(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

将一个raw SQL字符串添加到SELECT子句的选择列表中。

Adds a string of raw SQL to the select list of a SELECT clause.

命名空间： Appeon.Data

程序集： Appeon.Data.dll

语法

  ISqlQueryBuilder SelectRaw(string selectRaw, params ISqlParameter[] parameters);

Parameters

selectRaw System.String

A string of raw SQL added to the SELECT clause.

Actually, you can write any raw SQL if the SQL statement is valid.

parameters Appeon.Data.ISqlParameter[]

(Optional) An array of ISqlParameter objects which define the SQL parameters used in the raw SQL.

Returns

Appeon.Data.ISqlQueryBuilder

Returns the current ISqlQueryBuilder object.

Examples

演示SelectRaw(string selectRaw, params ISqlParam[] parameters)方法。以下案例设置自定义的列，并动态赋值

using Appeon.Data;
using System;

namespace Appeon.ApiDoc.ISqlQueryBuilderExamples
{
    public class SelectRawExample
    {
        private SchoolContext _context;

        public SelectRawExample(SchoolContext dataContext)
        {
            // Set Data Context
            _context = dataContext;
        }

        public void Example1()
        {
            // 声明SqlQueryBuilder
            var sqlbuilder = new SqlQueryBuilder();

            // 定义sql语法，设置自定义int类型的列selected
            sqlbuilder.SelectRaw("name, budget,selected = @a", 
                SqlBuilder.Parameter<int>("a"))
                .From("Department");

            // 将sqlbuilder语法转换成context对应数据库的可执行语句
            string sql = sqlbuilder.ToSqlString(_context);

            Console.WriteLine(sql);
            
            /*This code produces the following output:
            
            SELECT
            [name],
            [budget],
            [selected] = @a
            FROM [Department]
            */
        }
    }
}
