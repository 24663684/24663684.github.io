ISqlJoinBuilder 接口

.NET Standard 2.x | 当前版本(0.5.0-alpha)

0.5.0-alpha 0.5.1-alpha(current)
表示可用于在FROM子句中创建joined table的SQL Builder object，通过指定的联结操作将原始表源与新表源连接。

联结表是由两个或者更多表的积构成的结果集。

Represents a SQL Builder object which can be used to create a joined table in the FROM clause. Joins the original table source with a new table source by the specified JOIN operation.

A joined table is a result set that combines two of more tables.
