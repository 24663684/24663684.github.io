ISqlJoinBuilder.JoinRaw(string joinClause, string builderAlias) 方法

.NET Standard 2.x | 当前版本(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

不指定联接运算符的类型, 将原始表源直接联接到原始SQL字符串以创建连接表。 需要为原始SQL指定别名。

Uses the INNER JOIN operator to connect the original table source directly with a string of raw SQL, without specifying the type of the JOIN operator. You need to specify an alias for the raw SQL.

命名空间： Appeon.Data

程序集： Appeon.Data.dll

语法

  ISqlJoinOnBuilder JoinRaw(string joinClause, string builderAlias);

Parameters

joinClause System.String

A string of raw SQL after the JOIN keyword.

builderAlias System.String

The alias specified for the raw SQL.

Returns

Appeon.Data.ISqlJoinOnBuilder

Returns an ISqlJoinOnBuilder object which can be used to create the condition on which the join is based.

Examples

使用join将一个表与另外一表关联.

using Appeon.Data;
using System;

namespace Appeon.ApiDoc.ISqlJoinBuilderExamples
{
    public class JoinRawExample
    {
        private readonly SchoolContext _context;

        public JoinRawExample(SchoolContext dataContext)
        {
            // Sets the data context.
            _context = dataContext;
        }

        public void Example2()
        {
            var sqlQueryBuilder = new SqlQueryBuilder();
            
            // 定义一查询(命名为C), 获取Course表中每个不同DepartmentID的数量
            // 获取Department表中DepartmentID，Name 和以上查询表中数量
            // Department和以上查询表关联条件为Department.DepartmentID = C.DepartmentID
            sqlQueryBuilder
                .Select("Department.DepartmentID")
                .Select("Department.Name")
                .Select("C.CourseCount")
                .From("Department")
                .JoinRaw("(Select DepartmentID, count(1) as CourseCount From " +
                    "Course  group by DepartmentID ) C " +
                    "On Department.DepartmentID = C.DepartmentID"
                    , "Course");

            // 将sqlQueryBuilder语法转换成context对应数据库的可执行语句
            string sql = sqlQueryBuilder.ToSqlString(_context);

            Console.WriteLine(sql);
            
            /*This code example produces the following output:           
              SELECT
                [Department].[DepartmentID],
                [Department].[Name],
                [C].[CourseCount]
              FROM [Department]
              JOIN (SELECT
                        [DepartmentID],
                        COUNT(1) AS [CourseCount]
                    FROM [Course]
                    GROUP BY
                        [DepartmentID] ) [C]
              ON [Department].[DepartmentID] = [C].[DepartmentID]
            */
        }
    }
}
