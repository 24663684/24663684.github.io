ISqlJoinBuilder.AsWhere() 方法

.NET Standard 2.x | 当前版本(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

返回一个ISqlWhereAndOr对象，该对象表示一个用于在SQL语句中构建WHERE子句的SQL Builder对象。

Returns an ISqlWhereAndOr object which represents a SQL Builder object used to build the WHERE clause in a SQL statement.

命名空间： Appeon.Data

程序集： Appeon.Data.dll

语法

  ISqlWhereAndOr AsWhere();

Returns

Appeon.Data.ISqlWhereAndOr

Returns the ISqlWhereAndOr object which can be used to build the WHERE clause in a SQL statement (e.g.: adding search conditions to the WHERE clause).

Examples

在ISqlQueryBuilder中获取where 标识

using Appeon.Data;
using System;

namespace Appeon.ApiDoc.ISqlJoinBuilderExamples
{
    public class AsWhereExample
    {
        private readonly SchoolContext _context;

        public AsWhereExample(SchoolContext dataContext)
        {
            // Set data context
            _context = dataContext;
        }

        public void Example()
        {
            var sqlQueryBuilder = new SqlQueryBuilder();

            // 获取Department表中DepartmentID，Name数据，
            // 查询条件为DepartmentID < 5
            sqlQueryBuilder
                .Select("DepartmentID")
                .Select("Name")
                .From("Department")
                .WhereRaw("DepartmentID < 5");

            // 在以上查询条件上增加一个查询条件DepartmentID > 1
            sqlQueryBuilder.AsWhere().AndWhereRaw("DepartmentID > 1");

            // 将sqlQueryBuilder语法转换成context对应数据库的可执行语句
            string sql = sqlQueryBuilder.ToSqlString(_context);

            Console.WriteLine(sql);

            /*This code example produces the following output:           
               SELECT
                [DepartmentID],
                [Name]
               FROM[Department]
               WHERE([DepartmentID] < 5
                     AND[DepartmentID] > 1)
            */
        }
    }
}
