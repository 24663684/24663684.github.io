SqlBuilder.OnValue(string left, SqlBinaryOperator sqlOperator, object value) 方法

.NET Standard 2.x | 当前版本(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

创建一个ISqlJoinCondition对象, 表示在构造ON子句时可以使用的搜索条件，在运算符左侧指定SQL表达式, 在运算符右侧指定具体的值。

Creates an ISqlJoinCondition object which represents a search condition that can be used when building the ON clause. Specifies a SQL expression on the left the operator and a specific value on the right of the operator.

命名空间： Appeon.Data

程序集： Appeon.Data.dll

语法

 public static ISqlJoinCondition OnValue(string left, SqlBinaryOperator sqlOperator, object value);

Parameters

left System.String

An expression on the left of the operator.

SqlBinaryOperator Appeon.Data.SqlBinaryOperator

An enumeration value of SqlBinaryOperator, which is the operator to test the expression on the left and the value on the right.

value System.Object

A specific value on the right of the operator.

Returns

Appeon.Data.ISqlJoinCondition

Returns the ISqlJoinCondition object which can be used to add more search conditions to the ON clause.

Examples

演示OnValue(string left, SqlOperator sqlOperator, object value)以下案例演示Department表join Course表.使用OnValue()增加join条件;

using Appeon.Data;
using System;

namespace Appeon.ApiDoc.SqlBuilderExamples
{
    public class OnValueExample
    {
        private SchoolContext _context;

        public OnValueExample(SchoolContext dataContext)
        {
            // Set Data Context
            _context = dataContext;
        }

        public void Example2()
        {
            var sqlQueryBuilder = new SqlQueryBuilder();

            // 定义Join条件
            var join = SqlBuilder
                .OnValue("Department.DepartmentID", SqlBinaryOperator.Equals, 1);

            // 创建sql语法
            var query = sqlQueryBuilder
                .Select("Department.DepartmentID")
                .Select("Department.Name")
                .Select("Course.Title")
                .From("Department")
                .Join("Course")
                .On(join);

            // 转换成context对应数据库的可执行语句
            string sql = query.ToSqlString(_context);

            Console.WriteLine(sql);
            
            /*This code example produces the following output:           
               SELECT
                [Department].[DepartmentID],
                [Department].[Name],
                [Course].[Title]
                FROM [Department] JOIN [Course]
                ON [Department].[DepartmentID] = 1
            */
        }
    }
}
