SqlBuilder.OnRaw(string clause, params ISqlParameter[] parameters) 方法

.NET Standard 2.x | 当前版本(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

创建一个ISqlJoinCondition对象, 表示在构造ON子句时可以使用的搜索条件, 该搜索条件由一个raw SQL字符串指定。

Creates an ISqlJoinCondition object which represents a search condition that can be used when building the ON clause. This search condition is specified by a raw SQL string.

命名空间： Appeon.Data

程序集： Appeon.Data.dll

语法

 public static ISqlJoinCondition OnRaw(string clause, params ISqlParameter[] parameters);

Parameters

clause System.String

A string of raw SQL added to the ON clause.

parameters Appeon.Data.ISqlParameter[]

(Optional) An array of ISqlParameter objects which defines the SQL parameters used in the raw SQL.

Returns

Appeon.Data.ISqlJoinCondition

Returns the ISqlJoinCondition object which can be used to add more search conditions to the ON clause.

Examples

演示OnRaw(string clause, params ISqlParam[] parameters)以下案例演示Department表join Course表.join条件使用OnRaw()连接;parameters是可选参数

using Appeon.Data;
using System;

namespace Appeon.ApiDoc.SqlBuilderExamples
{
    public class OnRawExample
    {
        private SchoolContext _context;

        public OnRawExample(SchoolContext dataContext)
        {
            // Set Data Context
            _context = dataContext;
        }

        public void Example1()
        {
            var sqlQueryBuilder = new SqlQueryBuilder();

            // 定义Join条件
            var join = SqlBuilder
                .OnRaw("Department.DepartmentID = Course.DepartmentID and " +
                "Course.DepartmentID > @dept", 
                SqlBuilder.Parameter<int>("dept"));

            // 创建sql语法, 并转换成context对应数据库的可执行语句
            var query = sqlQueryBuilder
                .Select("Department.DepartmentID")
                .Select("Department.Name")
                .Select("Course.Title")
                .From("Department")
                .Join("Course")
                .On(join);

            // 转换成context对应数据库的可执行语句
            string sql = query.ToSqlString(_context);

            Console.WriteLine(sql);
            
            /*This code produces the following output:
            
            SELECT
            [Department].[DepartmentID],
            [Department].[Name],
            [Course].[Title]
            FROM [Department] JOIN [Course]
            ON ([Department].[DepartmentID] = [Course].[DepartmentID]
            AND [Course].[DepartmentID] > @dept)
            */
        }
    }
}
