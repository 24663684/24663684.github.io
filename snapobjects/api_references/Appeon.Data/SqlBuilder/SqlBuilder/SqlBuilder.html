SqlBuilder 类

.NET Standard 2.x | Current Version (0.5.0-alpha)

0.5.0-alpha 0.5.1-alpha (current)
SQLBuilder提供结构化的SQL语句构建方式，SQLBuilder只负责SQL语法结构的构建，并不依赖于具体的数据库，支持根据不同的DataContext生成不同数据库的SQL语句，使用SQLBuilder可以方便的动态构建SQL并将其缓存起来重复利用，还可以进行克隆和修改，解决了实际项目中大量SQL难以维护的难题。
