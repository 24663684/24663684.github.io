ISqlHavingAndOr<TBuilder>.AndHaving(string left, ISqlBuilder sqlBuilder) Method

.NET Standard 2.x | Current Version (0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha (current)

向HAVING子句添加AND逻辑运算符和一个搜索条件，在运算符左侧指定SQL表达式，在右侧指定SQL子查询。运算符使用'='。

Adds the AND logical operator and a search condition to the HAVING clause. Specifies a SQL expression on the left of the operator and a SQL subquery on the right of the operator. The operator is '='.

Namespace: Appeon.Data

Assembly: Appeon.Data.dll

Syntax

  TBuilder AndHaving(string left, ISqlBuilder sqlBuilder);

Parameters

left System.String

A SQL expression on the left of the operator.

sqlBuilder Appeon.Data.ISqlBuilder

An ISqlBuilder object which represents a SQL subquery on the right of the operator.

Returns

TBuilder

Returns the TBuilder object which can be used to add more search conditions to the current HAVING clause.

Examples

增加分组条件The following code example demonstrates how to specify search conditions for groups.

using Appeon.Data;
using System;

namespace Appeon.ApiDoc.ISqlHavingAndOrExamples
{
    public class AndHavingExample
    {
        private readonly SchoolContext _context;

        public AndHavingExample(SchoolContext dataContext)
        {
            // Set data context
            _context = dataContext;
        }

        public void Example5()
        {
            var sqlQueryBuilder = new SqlQueryBuilder();

            // 设置一个查询，用来获取最小Course ID
            // Sets a query to get the minimum Course ID.
            var sqlBuilder = new SqlQueryBuilder();
            sqlBuilder.SelectRaw("min(CourseID)")
                      .From("Course");

            // 在StudentGrade表中根据StudentID和CourseID分组，获取每个分组的Grade总和，
            // 分组条件为StudentID > 0，并且CourseID等于一个子查询的值
            // Groups the "StudentGrade" table by StudentID and CourseID, and gets the Grade sum of each group.
            // Specifies the search condition for the group as “StudentID > 0” and adds a HAVING condition: CourseID is equal to the return result of a subquery.
            sqlQueryBuilder
                .Select("StudentID")
                .Select("sum(Grade)")
                .From("StudentGrade")
                .GroupBy("StudentID, CourseID")
                .HavingRaw("StudentID > 0")
                .AndHaving("CourseID", sqlBuilder);

            // 将sqlQueryBuilderSyntax转换成context对应数据库的可执行语句
            // Returns the raw SQL for the database corresponding to the data context.
            string sql = sqlQueryBuilder.ToSqlString(_context);

            Console.WriteLine(sql);            

            /*This code example produces the following output:           
              SELECT
                [StudentID],
                sum(Grade)
              FROM [StudentGrade]
              GROUP BY
                [StudentID],
                [CourseID]
                HAVING ([StudentID] > 0
                AND [CourseID] = (SELECT
                                  MIN([CourseID])
                                  FROM [Course]))
           */
        }
    }
}
