ISqlHavingAndOr<TBuilder>.AndHavingRaw(string clause, string alias, params ISqlParameter[] parameters) Method

.NET Standard 2.x | Current Version (0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha (current)

向HAVING子句添加AND逻辑运算符和一个raw SQL字符串。需要未raw SQL指定别名。

Adds the AND logical operator and a raw SQL string to the HAVING clause. You need to specify an alias for the raw SQL.

Namespace: Appeon.Data

Assembly: Appeon.Data.dll

Syntax

  TBuilder AndHavingRaw(string clause, string alias, params ISqlParameter[] parameters);

Parameters

clause System.String

A string of raw SQL added to the HAVING clause.

alias System.String

The alias specified for the raw SQL.

parameters Appeon.Data.ISqlParameter

(Optional) An array of ISqlParameter objects which define the SQL parameters used in the raw SQL.

Returns

TBuilder

Returns the TBuilder object which can be used to add more search conditions to the current HAVING clause.

Examples

增加分组条件The following code example demonstrates how to specify search conditions for groups.

using Appeon.Data;
using System;

namespace Appeon.ApiDoc.ISqlHavingAndOrExamples
{
    public class AndHavingRawExample
    {
        private readonly SchoolContext _context;

        public AndHavingRawExample(SchoolContext dataContext)
        {
            // Set data context
            _context = dataContext;
        }

        public void Example2()
        {
            var sqlQueryBuilder = new SqlQueryBuilder();

            // 在StudentGrade表中根据StudentID和CourseID分组，获取每个分组的Grade总和，
            // 分组条件为StudentID > 0，并且StudentID小于一个输入参数
            // 输入参数在执行sql时指定
            // 将定义的分组条件起一个名称，供程序修改或者移除这个having条件
            // Groups the "StudentGrade" table by StudentID and CourseID, and gets the Grade sum of each group.
            // Specifies the search condition for the group as “StudentID > 0”, and adds a HAVING condition: StudentID is smaller than the input parameter.
            // The input parameter is specified when executing the SQL.
            // Gives the search condition a name to faciliate the program to modify or remove the HAVING condition.

            sqlQueryBuilder
                .Select("StudentID")
                .Select("sum(Grade)")
                .From("StudentGrade")
                .GroupBy("StudentID, CourseID")
                .HavingRaw("StudentID > 0")
                .AndHavingRaw("StudentID < @studentid", "having",
                              SqlBuilder.Parameter<int>("studentid"));

            // 并将sqlQueryBuilderSyntax转换成context对应数据库的可执行语句
            // Returns the raw SQL for the database corresponding to the data context.
            string sql = sqlQueryBuilder.ToSqlString(_context);

            Console.WriteLine(sql);

            // execute sql, 指定输入参数为10
            // Executes the SQL and specifies the input parameter as "10".     
            var studentGrade = _context.SqlExecutor.Select<DynamicModel>(sql, 10);

            /*This code example produces the following output:           
              SELECT
                [StudentID],
                sum(Grade)
              FROM [StudentGrade]
              GROUP BY
                [StudentID],
                [CourseID]
                HAVING ([StudentID] > 0
                AND [StudentID] < @studentid)
           */
        }
    }
}
