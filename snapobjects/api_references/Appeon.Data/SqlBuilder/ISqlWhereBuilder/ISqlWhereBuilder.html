代表一个用于构建SQL statement中的WHERE clause的SQL Builder object.

WHERE子句指定查询返回的行的搜索条件。

Represents a SQL Builder object used to build the WHERE clause in a SQL statement.

The WHERE clause specifies the search condition for the rows returned by a query.
