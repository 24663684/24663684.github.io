ISqlWhereBuilder.Where(string left, string right) 方法

.NET Standard 2.x | 当前版本(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

创建WHERE子句, 向WHERE子句添加搜索条件，在运算符左侧和右侧指定SQL表达式，运算符使用'='。

Creates a WHERE clause, and adds a search condition to the WHERE clause. Specifies SQL expressions on the left and right of the operator. The operator is '='.

命名空间： Appeon.Data

程序集： Appeon.Data.dll

语法

  ISqlWhereAndOr Where(string left, string right);

Parameters

left System.String

A SQL expression on the left of the operator.

right System.String

A SQL expression on the right of the operator.

Returns

Appeon.Data.ISqlWhereAndOr

Returns the ISqlWhereAndOr object which can be used to add more search conditions to the current WHERE clause.

Examples

多表查询时，设置匹配条件为两表字段相等

using Appeon.Data;
using System;

namespace Appeon.ApiDoc.ISqlWhereBuilderExamples
{
    public class WhereExample
    {
        private readonly SchoolContext _context;

        public WhereExample(SchoolContext dataContext)
        {
            // Set data context
            _context = dataContext;
        }

        public void Example1()
        {
            // 声明SqlQueryBuilder
            ISqlQueryBuilder sqlQueryBuilder = new SqlQueryBuilder();

            // 定义SQL语句， 并设置两表的关联列
            sqlQueryBuilder.Select("a.CourseID")
                           .Select("a.Title")
                           .From("Course", "a")
                           .From("Department", "b")
                           .Where("a.DepartmentID", "b.DepartmentID");

            // Show SQL
            Console.WriteLine("The SQL statement: \n{0}",
                             sqlQueryBuilder.ToSqlString(_context));

            // Execute sql
            var results =
                _context.SqlExecutor.Select<DynamicModel>(sqlQueryBuilder);

            /*This code produces the following output:

            The SQL statement: 
            SELECT
             [a].[CourseID],
            [a].[Title]
            FROM [Course] AS [a] ,
            [Department] AS [b]
            WHERE ([a].[DepartmentID] = [b].[DepartmentID])
           */
        }
    }
}
