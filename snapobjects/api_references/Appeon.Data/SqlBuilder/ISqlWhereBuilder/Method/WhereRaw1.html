ISqlWhereBuilder.WhereRaw(string whereClause, params ISqlParameter[] parameters) 方法

.NET Standard 2.x | 当前版本(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

创建WHERE子句, 向WHERE子句添加raw SQL。

Creates a WHERE clause, and adds an raw SQL to the WHERE clause.

命名空间： Appeon.Data

程序集： Appeon.Data.dll

语法

  ISqlWhereAndOr WhereRaw(string whereClause, params ISqlParameter[] parameters);

Parameters

whereClause System.String

A string of raw SQL added to the WHERE clause.

parameters Appeon.Data.ISqlParameter[]

(Optional) An array of ISqlParameter objects which defines the SQL parameters used in the raw SQL.

Returns

Appeon.Data.ISqlWhereAndOr

Returns the ISqlWhereAndOr object which can be used to add more search conditions to the current WHERE clause.

Examples

以下案例查询Course表中满足Title = @titlename(参数)的记录使用原生态的SQL语句构造where条件

using Appeon.Data;
using System;

namespace Appeon.ApiDoc.ISqlWhereBuilderExamples
{
    public class WhereRawExample
    {
        private readonly SchoolContext _context;

        public WhereRawExample(SchoolContext dataContext)
        {
            // Set data context
            _context = dataContext;
        }

        public void Example1()
        {
            // 声明SqlQueryBuilder
            var sqlbuilder = new SqlQueryBuilder();

            // 定义SQL语句，并使用原生态的SQL语句构造查询条件:Title = @titlename
            sqlbuilder.Select("*")
                .From("Course")
                .WhereRaw("Title = @titlename", 
                           SqlBuilder.Parameter<string>("titlename"));

            string sql = sqlbuilder.ToSqlString(_context);

            Console.WriteLine(sql);

            /*This code produces the following output:

            SELECT
            *
            FROM [Course]
            WHERE ([Title] = @titlename)
           */
        }
    }
}
