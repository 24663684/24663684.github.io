ISqlFromBuilder.FromRaw(string fromClause) Method

.NET Standard 2.x | Current Version (0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha (current)

向当前SQL Builder对象中的FROM从句添加一个(由raw SQL指定的) 表源(表，视图或者派生表)。

Adds a table source (table, view or derived table), specified by a raw SQL, to the FROM clause of the current SQL Builder object.

Namespace: Appeon.Data

Assembly: Appeon.Data.dll

Syntax

  ISqlFromBuilder FromRaw(string fromClause);

Parameters

fromClause System.String

A string of raw SQL which is a table source (table, view or derived table).

Actually, you can write any raw SQL if the SQL statement is valid.

Returns

Appeon.Data.ISqlFromBuilder

Returns the current ISqlFromBuilder object.

Examples

演示FromRaw(string fromClause)方法。以下案例从子查询中查询所有记录The following code example demonstrates how to use the FromRaw(string fromClause) method.It creates a derived table by a subquery and retrieves all the records from the derived table.

using Appeon.Data;
using System;

namespace Appeon.ApiDoc.ISqlFromBuilderExamples
{
    public class FromRawExample
    {
        private SchoolContext _context;

        public FromRawExample(SchoolContext dataContext)
        {
            // Set Data Context
            _context = dataContext;
        }

        public void Example1()
        {
            // 声明SqlQueryBuilder
            // Declares SqlQueryBuilder.
            var sqlbuilder = new SqlQueryBuilder();

            // 定义SQL语句，创建子查询，并从中查询所有记录
            // Defines a SQL statement to create a derived table by a subquery and retrieve all the records from the derived table.
            sqlbuilder.Select("*")
                .FromRaw("(select name, budget from department) as dept");

            // 将sqlbuilderSyntax转换成context对应数据库的可执行语句
            // Returns the raw SQL for the database corresponding to the data context.
            string sql = sqlbuilder.ToSqlString(_context);

            Console.WriteLine(sql);
            
            /*This code produces the following output:
            
            SELECT
            *
            FROM (SELECT
            [name],
            [budget]
            FROM [department]) AS [dept]      
            */
        }
    }
}
