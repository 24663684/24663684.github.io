ISqlWhereAndOr<TBuilder>.OrWhereValue(string left, object value) 方法

.NET Standard 2.x | 当前版本(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

向WHERE子句添加OR逻辑运算符和一个搜索条件，在运算符左侧指定SQL表达式, 在运算符右侧指定具体的值。运算符使用 '='。

Adds the OR logical operator and a search condition to the WHERE clause. Specifies a SQL expression on the left the operator and a specific value on the right of the operator. The operator is '='.

命名空间： Appeon.Data

程序集： Appeon.Data.dll

语法

  TBuilder OrWhereValue(string left, object value);

Parameters

left System.String

An expression on the left of the operator.

value System.Object

The specific value on the right of the operator.

Returns

TBuilder

Returns the TBuilder object which can be used to add more search conditions to the current WHERE clause.

Examples

以下案例查询Course表中满足CourseID=@courid OR Title = 'Physics'的记录判断指定的列是否与查询参数设置值相等

using Appeon.Data;
using System;

namespace Appeon.ApiDoc.ISqlWhereAndOrExamples
{
    public class OrWhereValueExample
    {
        private readonly SchoolContext _context;

        public OrWhereValueExample(SchoolContext dataContext)
        {
            // Set data context
            _context = dataContext;
        }

        public void Example1()
        {
            // 声明SqlQueryBuilder
            var sqlbuilder = new SqlQueryBuilder();

            // 定义SQL语句， 并设置查询条件：OR Title = 'Physics'
            // 判断指定的列Title是否与参数值'Physics'相等
            sqlbuilder.Select("*")
                .From("Course")
                .Where("CourseID", SqlBuilder.Parameter<int>("courid"))
                .OrWhereValue("Title", "Physics");

            string sql = sqlbuilder.ToSqlString(_context);

            Console.WriteLine(sql);

            /*This code produces the following output:

            SELECT
            *
            FROM [Course]
            WHERE ([CourseID] = @courid
            OR [Title] = N'Physics')
           */
        }
    }
}
