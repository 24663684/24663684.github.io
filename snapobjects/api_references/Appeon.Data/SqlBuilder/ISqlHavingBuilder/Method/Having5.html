ISqlHavingBuilder.Having(string left, ISqlBuilder sqlBuilder) 方法

.NET Standard 2.x | 当前版本(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

创建HAVING子句,向HAVING子句添加搜索条件，在运算符左侧指定SQL表达式，在右侧指定SQL子查询。运算符使用'='。

Creates a HAVING clause and adds a search condition to the HAVING clause. Specifies a SQL expression on the left of the operator and specifies a SQL subquery on the right side. The operator is '='.

命名空间： Appeon.Data

程序集： Appeon.Data.dll

语法

  ISqlHavingAndOr Having(string left, ISqlBuilder sqlBuilder);

Parameters

left System.String

A SQL expression on the left of the operator.

sqlBuilder Appeon.Data.ISqlBuilder

An ISqlBuilder object which represents a SQL subquery on the right of the operator.

Returns

Appeon.Data.ISqlHavingAndOr

Returns the ISqlHavingAndOr object which can be used to add more search conditions to the current HAVING clause.

Examples

设置分组条件The following code example demonstrates how to specify search conditions for groups.

using Appeon.Data;
using System;

namespace Appeon.ApiDoc.ISqlHavingBuilderExamples
{
    public class HavingExample
    {
        private readonly SchoolContext _context;

        public HavingExample(SchoolContext dataContext)
        {
            // Sets the data context.
            _context = dataContext;
        }

        public void Example5()
        {
            var sqlQueryBuilder = new SqlQueryBuilder();

            // 设置一个查询，用来获取最小Course ID
            // Sets a query to get the minimum Course ID.
            var sqlBuilder = new SqlQueryBuilder();
            sqlBuilder.SelectRaw("min(CourseID)")
                      .From("Course");

            // 在StudentGrade表中根据StudentID和CourseID分组，获取每个分组的Grade总和，
            // CourseID等于一个子查询的值
            // Groups the "StudentGrade" table by StudentID and CourseID, and gets the Grade sum of each group. 
            // CourseID is equal to the return value of a subquery.        
            sqlQueryBuilder
                .Select("StudentID")
                .Select("sum(Grade)")
                .From("StudentGrade")
                .GroupBy("StudentID, CourseID")
                .Having("CourseID", sqlBuilder);

            // 将sqlQueryBuilder语法转换成context对应数据库的可执行语句
            // Returns the raw SQL for the database corresponding to the data context.              
            string sql = sqlQueryBuilder.ToSqlString(_context);

            Console.WriteLine(sql);

            /*This code example produces the following output:           
              SELECT
                [StudentID],
                sum(Grade)
              FROM [StudentGrade]
              GROUP BY
                [StudentID],
                [CourseID]
              HAVING ( [CourseID] = (SELECT
                       MIN([CourseID])
                       FROM [Course]))
           */
        }
    }
}
