IModelMapper.TrackSqlBuilder(ISqlDeleteBuilder deleteBuilder, params object[] parameters) Method

.NET Standard 2.x | Current version (0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha (current)

Tracks a database table delete operation. A SQL DELETE statement will be generated using the specified ISqlDeleteBuilder object. When IModelMapper.SaveChanges method is called, the SQL DELETE statement will be executed.

Namespace: Appeon.Data

Assembly: Appeon.Data.dll

Syntax

  public IModelMapper TrackSqlBuilder(ISqlDeleteBuilder deleteBuilder, params object[] parameters);

Parameters

deleteBuilder Appeon.Data.ISqlDeleteBuilder

An interface used to build the SQL DELETE statement.

parameters System.Object[]

(Optional) One or more values that you want to use as arguments in the DELETE SQL statement defined in deleteBuilder.

Returns

Appeon.Data.IModelMapper

Returns an IModelMapper object, which can be used for executing other methods .

Examples

The following code example demonstrates how to delete a student.

using Appeon.Data;
using System;

namespace Appeon.ApiDoc.IModelMapperExamples
{
    public class TrackSqlBuilderExample
    {
        private SchoolContext _context;

        public TrackSqlBuilderExample(SchoolContext dataContext)
        {
            // Sets the data context.
            _context = dataContext;
        }

        public void Example2()
        {

            var deleteBuilder = new SqlDeleteBuilder();

            // Builds a SQL Delete statement with no parameters to delete a student.
            deleteBuilder.Delete("Person")
                        .From("Person")
                        .Where("LastName", "'Senior'")
                        .AndWhere("FirstName", "'Letitia'");

            Console.WriteLine("SQL Delete statement is:");
            Console.WriteLine(deleteBuilder.ToSqlString(_context));

            // Tracks the SQL Delete statement and executes it in the database.
            var dbResult = _context.ModelMapper.TrackSqlBuilder(deleteBuilder)
                                               .SaveChanges();

            /* The code produces the following output:
            SQL Delete statement is:
            DELETE [Person]
            FROM [Person]
            WHERE ([LastName] = 'Senior'
            AND [FirstName] = 'Letitia')
            */
        }
    }
}
