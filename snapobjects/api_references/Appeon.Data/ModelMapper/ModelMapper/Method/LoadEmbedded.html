获取一个可以用于加载TModel实例中的嵌套属性的接口IEmbeddedLoadable。

Obtains the IEmbeddedLoadable interface which can be used to load the embedded property of a TModel instance.

Namespace: Appeon.Data

Assembly: Appeon.Data.dll

Syntax

  public IEmbeddedLoadable<TModel> LoadEmbedded<TModel>(TModel model, params object[] parameters);

Type Parameters

TModel

待定。

Parameters

model TModel

一个定义了一个或者多个嵌套属性的TModel实例。

A TModel instance which has defined one or more embedded properties.

parameters System.Object[]

One or more values that you want to use as retrieval arguments in the SQL SELECT statement defined in TModel.

这些参量并不会被用于检索TModel,而是仅用于处理嵌套属性，因为嵌套属性的定义中有可能引用到TModel中的参数。

These parameters will not be used to retrieve TModel, but used to process the embedded property, because the embedded property may have reference to these TModel parameters.

Returns

Appeon.Data.IEmbeddedLoadable<TModel>

返回一个可以加载TModel实例中的嵌套属性的接口，利用该接口的方法，可以对一个或者多个嵌套属性进行加载。

Returns an interface that can load the embedded property of a TModel instance. The method of this interface can load one or more embedded property.

备注

该方法并未对主Model或者其嵌套属性进行实际的检索，需要通过该方法返回的接口IEmbeddedLoadable<TModel>的具体方法数据检索和加载。

This method does not actually retrieve the main Model or its embedded property, instead it returns the IEmbeddedLoadable<TModel> interface whose methods will be used to retrieve and load data.

如果需要对主Model进行检索并加载数据，请使用Load<TModel>方法。

To retrieve and load data of the main Model, use the Load<TModel> method.
