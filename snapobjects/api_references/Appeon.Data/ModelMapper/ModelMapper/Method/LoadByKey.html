ModelMapper.LoadByKey<TModel>(params object[] parameters) 方法

.NET Standard 2.x | 当前版本(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

根据TModel的主表的定义及其主键检索数据。

Loads the data retrieved according to the mapped table and primary key defined in a TModel.

命名空间： Appeon.Data

程序集： Appeon.Data.dll

语法

  public ILoadable<TModel> LoadByKey<TModel>(params object[] parameters);

Type Parameters

TModel

待定。

Parameters

parameters System.Object[]

One or more values that you want to use as retrieval arguments in the SQL SELECT statement defined by the main table and it's primary key(s) in TModel.

Returns

Appeon.Data.ILoadable<TModel>

返回ILoadable<TModel>接口，可使用该接口的方法进一步获取数据结果集，或者进行嵌套查询等操作。

Returns the interface of ILoadable<TModel> which can be used to further obtain the data result set or perform data operations such as embedded queries.

备注

如果TModel中定义的SQL查询是多表查询，那么该方法仅会检索并得到主表的数据。

If the SQL query defined in the TModel makes query from multiple tables, then this method only retrieves and loads data from the mapped table.
