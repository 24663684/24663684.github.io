ModelStore<TModel>.ModelStore( ChangeTrackingStrategy strategy = ChangeTrackingStrategy.NotTrack**) Constructor**

.NET Standard 2.x | Current Version (0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha (current)

Initializes a new instance of the ModelStore class. You can specify the change tracking strategy. Data changes are not tracked by default.

Namespace: Appeon.Data

Assembly: Appeon.Data.dll

Syntax

  public ModelStore(ChangeTrackingStrategy strategy = ChangeTrackingStrategy.NotTrack);

Parameters

strategy Appeon.Data.ChangeTrackingStrategy

(Optional) A value of the ChangeTrackingStrategy enumerated datatype. The default is ChangeTrackingStrategy.NotTrack.

Examples

The following code example initializes a ModelStore object and specifies the tracking strategy.

using Appeon.ApiDoc.Models.School;
using Appeon.Data;
using System;
using System.Collections.Generic;

namespace Appeon.ApiDoc.ModelStoreExamples
{
    public class ModelStoreExample
    {
        private SchoolContext _context;

        public ModelStoreExample(SchoolContext dataContext)
        {
            // Sets the data context
            _context = dataContext;
        }

        public void Example2()
        {
            // Initializes a ModelStore object
            // Specifies the tracking strategy to PropertyState
            IModelStore<Course> model = 
                new ModelStore<Course>(ChangeTrackingStrategy.PropertyState);

            // Shows the tracking strategy
            Console.WriteLine("Tracking Strategy = {0}.",
                model.ChangeTrackingStrategy);

            /*This code produces the following output:

            Tracking Strategy = PropertyState.
           */
        }
    }
}
