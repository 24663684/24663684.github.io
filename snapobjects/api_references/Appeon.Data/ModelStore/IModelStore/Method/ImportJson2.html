IModelStore<TModel>.ImportJson(string json, int startRow) Method

.NET Standard 2.x | Current Version (0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha (current)

Inserts data from a JSON string into ModelStore. You can specify the starting position in the JSON array. Data before the starting position will not be imported into ModelStore.

Namespace: Appeon.Data

Assembly: Appeon.Data.dll

Syntax

  public int ImportJson(string json, int startRow);

Parameters

json System.String

A string specifying the JSON data.

The JSON string must comply with the simple-format JSON or standard-format JSON.

startRow System.Int32

The zero-based number of the first detail object in the JSON array that you want to import. If it is negative, 0 is used.

Returns

System.Int32

Returns the number of model objects that were imported if it succeeds.

Examples

The following code example demonstrates how to use the ImportJson(String, Int32) method to import the department data from the starting detail object in a JSON string to ModelStore.

using Appeon.ApiDoc.Models.School;
using Appeon.Data;
using System;
using System.IO;

namespace Appeon.ApiDoc.IModelStoreExamples
{
    public class ImportJsonExample
    {
        private SchoolContext _context;

        public ImportJsonExample(SchoolContext dataContext)
        {
            // Sets the data context
            _context = dataContext;
        }

        public void Example2()
        {
            // Creates ModelStore.
            var depts = new ModelStore<Department>();

            // Gets the JSON file.
            // ImportJson2.json contains 4 records.
            string path = Path.Combine(AppContext.BaseDirectory,
                                       @"Jsons\IModelStore\ImportJson2.json");
            string json = File.ReadAllText(path);

            // Imports data (starting from the first detail object to the end) from JSON.
            depts.ImportJson(json, 0);

            Console.WriteLine("Count: {0}", depts.Count);

            depts.Clear();

            // Imports data (starting from the second detail object to the end) from JSON.
            depts.ImportJson(json, 1);

            Console.WriteLine("Count: {0}", depts.Count);

            /* This code produces the following output:

            Count: 4
            Count: 3
           */
        }
    }
}
