IModelStore<TModel>.BinarySearch(Func<TModel, object> searchProperty, object propertyValue) Method

.NET Standard 2.x | Current Version (0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha (current)

Searches for the specified model object in ModelStore by using a binary search algorithm.

Namespace: Appeon.Data

Assembly: Appeon.Data.dll

Syntax

  public int BinarySearch(Func<TModel, object> searchProperty, object propertyValue);

Parameters

searchProperty System.Func<TModel,System.Object>

A transform function that will be invoked on the model object of ModelStore.

propertyValue System.Object

The value which will be used to compare with the result of the transform function.

Returns

System.Int32

Returns the zero-based index of the model object found in ModelStore.

Returns -1 if no data item was found.

Remarks

ModelStore must be sorted before using the BinarySearch method.

Examples

The following code example demonstrates how to use the BinarySearch method to search for the specified model object in ModelStore.

using Appeon.Data;
using Appeon.ApiDoc.Models.School;
using System;

namespace Appeon.ApiDoc.IModelStoreExamples
{
    public class BinarySearchExample
    {
        private readonly SchoolContext _context;

        public BinarySearchExample(SchoolContext dataContext)
        {
            // Sets the data context
            _context = dataContext;
        }

        public void Example()
        {
            var cources = new ModelStore<Course>();

            // Retrieves data.
            cources.Retrieve(_context);

            //ModelStore must be sorted before using the BinarySearch method.
            cources.Sort(m => m.CourseID);

            // Searches for the model object whose CourseID is 3141.
            int index = cources.BinarySearch(m => m.CourseID, 3141);

            Console.WriteLine("Index: {0}", index);

            /* This code example produces the following output:
            
            Index: 6
            */
        }
    }
}
