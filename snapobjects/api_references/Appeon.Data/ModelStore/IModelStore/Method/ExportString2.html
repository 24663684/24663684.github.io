IModelStore<TModel>.ExportString(int startRow, int endRow, bool colheading = true) Method

.NET Standard 2.x | Current Version (0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha (current)

Exports data from the model objects (from the specified range) of ModelStore to a string. The property names of TModel are exported by default.

Namespace: Appeon.Data

Assembly: Appeon.Data.dll

Syntax

  public string ExportString(int startRow, int endRow, bool colheading = true);

Parameters

startRow System.Int32

The zero-based index for the starting position in ModelStore.

endRow System.Int32

The zero-based index for the ending position in ModelStore. If it is negative, it indicates the rest of model objects.

colheading System.Boolean

True (default): to export the property names of TModel.

False: Not to export the property names of TModel.

Returns

System.String

Returns the string if it succeeds.

Remarks

Values are separated by tabs. A newline character ("\r\n" for non-Unix platforms and "\n" for Unix platforms) is placed at the end of each line.

Examples

The following code example demonstrates how to use the ExportString(Int32, Int32, Boolean) method to export the first two department records (with and without property names) from ModelStore to a string.

using Appeon.ApiDoc.Models.School;
using Appeon.Data;
using Appeon.Data.PowerBuilder;
using System;

namespace Appeon.ApiDoc.IModelStoreExamples
{
    public class ExportStringExample
    {
        private SchoolContext _context;

        public ExportStringExample(SchoolContext dataContext)
        {
            // Sets the data context
            _context = dataContext;
        }

        public void Example2()
        {
            // Creates ModelStore and retrieves data
            var department = new ModelStore<Department>();
            department.Retrieve(_context);

            // Exports data of model objects (starts at index 0 and ends at index 1) 
            // to a string with property names.
            string str = department.ExportString(0, 1);

            Console.WriteLine("ModelStore.ExportString(0, 1):");
            Console.WriteLine(str);

            // Exports data of model objects (starts at index 0 and ends at index 1) 
            // to a string without property names.
            str = department.ExportString(0, 1, false);

            Console.WriteLine("ModelStore.ExportString(0, 1, false):");
            Console.WriteLine(str);

            /* This code produces the following output:

           ModelStore.ExportString(0, 1):
            "Departmentid"  "Name"  "Budget"        "Startdate"     "Administrator"
            2       "English"       120000.0000     9/1/2007 12:00:00 AM    6
            4       "Economics"     200000.0000     9/1/2007 12:00:00 AM    4

            ModelStore.ExportString(0, 1, false):
            2       "English"       120000.0000     9/1/2007 12:00:00 AM    6
            4       "Economics"     200000.0000     9/1/2007 12:00:00 AM    4 
           */
        }
    }
}
