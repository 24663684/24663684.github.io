IModelStore<TModel>.GetPropertyState(int index, string propertyName) Method

.NET Standard 2.x | Current Version (0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha (current)

Gets the current tracked state of the specified property (by name) of the specified model object (by index) in ModelStore.

Namespace: Appeon.Data

Assembly: Appeon.Data.dll

Syntax

  public PropertyState GetPropertyState(int index, string propertyName);

Parameters

index System.Int32

The zero-based index in ModelStore from which to locate the model object.

propertyName System.String

The name of the property in the model object from which to get the property state.

Returns

Appeon.Data.PropertyState

Returns a value of the PropertyState enumerated datatype.

Examples

The following code example demonstrates how to get the tracked state of the specified property (by name) of the specified model object.

using Appeon.Data;
using Appeon.ApiDoc.Models.School;
using System;

namespace Appeon.ApiDoc.IModelStoreExamples
{
    public class GetPropertyStateExample
    {
        private readonly SchoolContext _context;

        public GetPropertyStateExample(SchoolContext dataContext)
        {
            // Sets the data context
            _context = dataContext;
        }

        public void Example2()
        {
            // Sets the tracking strategy to PropertyState.
            var depts =
                new ModelStore<Department>(ChangeTrackingStrategy.PropertyState);

            // Retrieves data.
            depts.Retrieve(_context);

            // Modifies the value of the Name property of model object at index 1.
            depts.SetValue(1, "Name", "New Department");

            // Gets the property state of the "Name" property of model object at 
            // index 1.
            var state = depts.GetPropertyState(1, "Name");

            Console.WriteLine("Property State: {0}", state);

            /* This code example produces the following output:
            
             Property State: Modified
            */
        }
    }
}
