IModelStore<TModel>.RemoveAt(int index) Method

.NET Standard 2.x | Current Version (0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha (current)

Removes the specified model object (by index) from ModelStore.

Namespace: Appeon.Data

Assembly: Appeon.Data.dll

Syntax

  public void RemoveAt(int index);

Parameters

index System.Int32

The zero-based index of the model object to remove.

Remarks

If ModelStore is being tracked, the removed model object will have the ModelState.Deleted state. You can call TrackedCount(StateTrackable.Deleted) to get the number of the tracked model object in ModelState.Deleted state.

Examples

The following code example demonstrates how to remove the model object whose index is 1 in ModeStore.

using Appeon.Data;
using Appeon.ApiDoc.Models.School;
using System;

namespace Appeon.ApiDoc.IModelStoreExamples
{
    public class RemoveAtExample
    {
        private readonly SchoolContext _context;

        public RemoveAtExample(SchoolContext dataContext)
        {
            // Sets the data context
            _context = dataContext;
        }

        public void Example()
        {
            var courses = new ModelStore<Course>();

            // Retrieves data.
            courses.Retrieve(_context);

            // Shows the number of model objects in ModelStore before removed.
            Console.WriteLine("Before Remove Count: {0}", courses.Count);

            // Removes the model object whose index is 1.
            courses.RemoveAt(1);

            // Shows the number of model objects in ModelStore after removed.
            Console.WriteLine("After Remove Count: {0}", courses.Count);

            /* This code example produces the following output:
            
            Before Remove Count: 12
            After Remove Count: 11
            */
        }
    }
}
