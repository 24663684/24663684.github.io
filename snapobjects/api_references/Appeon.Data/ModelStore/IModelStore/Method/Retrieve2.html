IModelStore<TModel>.Retrieve(DataContext context, ISqlQueryBuilder sqlBuilder, params object[] parameters) Method

.NET Standard 2.x | Current Version (0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha (current)

Retrieves data from the database according to the SQL SELECT statement defined in the ISqlQueryBuilder object. If there are SQL parameters defined in the ISqlQueryBuilder object, the corresponding arguments should be passed in.

Namespace: Appeon.Data

Assembly: Appeon.Data.dll

Syntax

  public int Retrieve(DataContext context, ISqlQueryBuilder sqlBuilder, params object[] parameters);

Parameters

context Appeon.Data.DataContext

A DataContext object which contains the database connection information.

sqlBuilder Appeon.Data.ISqlQueryBuilder

An ISqlQueryBuilder object used to build the SQL SELECT statement dynamically. The result set of this SQL SELECT statement should have the same column names and data types as the properties of TModel for the current ModelStore.

parameters System.Object

(Optional) One or more values that you want to use as retrieval arguments in the SQL SELECT statement defined in the ISqlQueryBuilder object. See ParamValue for more info.

Returns

System.Int32

Returns the number of model objects retrieved from database.

Examples

The following code example demonstrates how to retrieve data according to the SQL SELECT statement and the retrieval arguments defined in the ISqlQueryBuilder object.

using Appeon.Data;
using Appeon.ApiDoc.Models.School;
using System;

namespace Appeon.ApiDoc.IModelStoreExamples
{
    public class RetrieveExample
    {
        private readonly SchoolContext _context;

        public RetrieveExample(SchoolContext dataContext)
        {
            // Sets the data context
            _context = dataContext;
        }

        public void Example2()
        {
            var courses = new ModelStore<Course>();

            // Builds the SQL Query.
            ISqlQueryBuilder querySql = new SqlQueryBuilder();
            querySql.Select("CourseID")
                    .Select("Title")
                    .From("Course")
                    .Where("Courseid", SqlBinaryOperator.Equals,
                            SqlBuilder.Parameter(typeof(int)));

            // Retrieves data whose CourseID = 1.
            int count = courses.Retrieve(_context, querySql, 1);

            // Shows the number of mode objects in ModelStore.
            Console.WriteLine("Count: {0}", count);

            /* This code example produces the following output:
            
            Count: 1
            */
        }
    }
}
