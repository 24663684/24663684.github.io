IModelStore<TModel>.BinarySearch(Func<TModel, object> searchProperty, object propertyValue) Method

使用二分查找算法在ModelStore中查找特定model object。

Uses a binary search algorithm to locate a specific model object in the ModelStore.

Namespace: Appeon.Data

Assembly: Appeon.Data.dll

Syntax

  public int BinarySearch(Func<TModel, object> searchProperty, object propertyValue);

Parameters

searchProperty System.Func<TModel,System.Object>

A transform function that will be invoked on the model objects of ModelStore.

propertyValue System.Object

The value which will be used to compare with each result of the transform function.

Returns

System.Int32

Returns the zero-base index of the model object found in the ModelStore.

Returns -1 if no data item found.

Remarks

Examples

使用二分搜索方式，按设置的查找条件对Model中的数据进行查找。

using Appeon.Data;
using Appeon.ApiDoc.Models.School;
using System;

namespace Appeon.ApiDoc.IModelStoreExamples
{
    public class BinarySearchExample
    {
        private readonly SchoolContext _context;

        public BinarySearchExample(SchoolContext dataContext)
        {
            // Set data context
            _context = dataContext;
        }

        public void Example()
        {
            IModelStore<Department> dept = new ModelStore<Department>();

            //检索出Department中的数据
            dept.Retrieve(_context);

            //对检索出的数据进行排序
            //使用BinarySearch前，必须对Model进行排序
            dept.Sort(m => m.DepartmentID);

            //查找Model中，DepartmentID = 10的记录，返回值为行号 
            int row = dept.BinarySearch(m => m.DepartmentID, 10);

            Console.WriteLine("row = {0}", row);

            /* This code example produces the following output:
            
             row = 1
            */
        }
    }
}
