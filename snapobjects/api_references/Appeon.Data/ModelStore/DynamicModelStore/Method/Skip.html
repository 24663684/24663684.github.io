IModelStore<TModel>.Skip(int count) Method

.NET Standard 2.x | Current Version(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha (current)

跳过ModelStore中指定数量的model objects，然后返回一个新的ModelStore，该ModelStore由剩余的model objects创建的。

Bypasses a specified number of model object in the ModelStore and then returns a new ModelStore created the remaining model objects.

Namespace: Appeon.Data

Assembly: Appeon.Data.dll

Syntax

  public IModelStore<TModel> Skip(int count);

Parameters

count System.Int32

The number of model objects to skip before returning the remaining model objects.

Returns

Appeon.Data.IModelStore<TModel>

If the ModelStore is being tracked, returns a new ModelStore which is created by the clones of remaining model objects. Otherwise, returns a new ModelStore which is created by the remaining model objects directly.

Remarks

Examples

跳过Model中指定的记录行数，返回剩余记录

using Appeon.Data;
using Appeon.ApiDoc.Models.School;
using System;

namespace Appeon.ApiDoc.IModelStoreExamples
{
    public class SkipExample
    {
        private readonly SchoolContext _context;

        public SkipExample(SchoolContext dataContext)
        {
            // Set data context
            _context = dataContext;
        }

        public void Example()
        {
            IModelStore<Course> courses = new ModelStore<Course>();

            // 检索出courses中的数据
            courses.Retrieve(_context);

            // 输出Model中的记录行数
            Console.WriteLine("Before Skip:");
            Console.WriteLine("Row Count = {0}", courses.Count);

            // 跳过Model中前两行记录
            // 返回Model中剩余的记录
            var skipModel = courses.Skip(2);

            // 输出执行Skip后返回的记录行数
            Console.WriteLine("After Skip:");
            Console.WriteLine("Row Count = {0}", skipModel.Count);

            /* This code example produces the following output:
            
            Before Skip:
            Row Count = 11
            After Skip:
            Row Count = 9
            */
        }
    }
}
