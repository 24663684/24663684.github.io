IModelStore<TModel>.Filter(Func<TModel, bool> predicate, bool removeFiltered = false) Method

.NET Standard 2.x | Current Version(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha (current)

获取一个新的ModelStore,它由当前ModelStore中过滤条件匹配的model objects创建, 可以选择是否将满足过滤的model objects从当前ModelStore中移除。

Gets a new ModelStore created by the model objects, which match the filter criteria in the current ModelStore . You can choose whether to remove the filtered model objects from the current ModelStore.

Namespace: Appeon.Data

Assembly: Appeon.Data.dll

Syntax

  public IModelStore<TModel> Filter(Func<TModel, bool> predicate, bool removeFiltered = false);

Parameters

predicate System.Func<TModel, System.Boolean>

An Func<TModel, bool> which specifies the filter criteria.

removeFiltered System.Boolean

True: remove the filtered model objects from the current ModelStore.

False: don't remove the filtered model objects from the current ModelStore.

Default is false.

Returns

Appeon.Data.IModelStore<TModel>

If the current ModelStore is being tracked, and removeFiltered is false, returns the new ModelStore created by the clones of the filtered model objects;

Otherwise, returns the new ModelStore created by the filtered model objects directly.

Remarks

Examples

按指定条件，过滤出Model中的数据。

using Appeon.Data;
using Appeon.ApiDoc.Models.School;
using System;

namespace Appeon.ApiDoc.IModelStoreExamples
{
    public class FilterExample
    {
        private readonly SchoolContext _context;

        public FilterExample(SchoolContext dataContext)
        {
            // Set data context
            _context = dataContext;
        }

        public void Example()
        {
            IModelStore<Department> dept = new ModelStore<Department>();

            //检索出Department中的数据
            dept.Retrieve(_context);            

            //过滤Model中，DepartmentID = 2的记录
            var filterModel = dept.Filter(m => m.DepartmentID == 2);

            Console.WriteLine("Filter Count = {0}", filterModel.Count);

            /* This code example produces the following output:
            
            Filter Count = 1
            */
        }
    }
}
