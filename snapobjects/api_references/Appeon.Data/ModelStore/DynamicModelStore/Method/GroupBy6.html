IDataStore.GroupBy<TModel>(string selectorKey, string sortKey = null) Method

.NET Standard 2.x | Current Version(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha (current)

泛型Method，按自定义分组条件对主缓冲区的行进行分组。 可以指定排序条件。

Generic method. Groups rows of the primary buffer by the custom grouping criteria. The sort criteria can be specified.

Namespace: Appeon.Data.PowerBuilder

Assembly: Appeon.Data.PowerBuilder.dll

Syntax

  public IEnumerable<IGrouping<object, TModel>> GroupBy<TModel>(string selectorKey, string sortKey = null);

Type Parameters

TModel

The type of a model class that conforms to the current DataObject .

Parameters

selectorkey System.String

A comma-separated list of the columns or expressions that control the grouping of the data.

sortkey System.String

A string whose value is the valid sort criteria for the DataWindow. The expression includes column names or numbers. A column number must be preceded by a pound sign (#).

Default is null which indicates it will use the current sort criteria in DataStore.

Returns

System.Collections.Generic.IEnumerable<IGrouping<System.Object, Appeon.Data.PowerBuilder.IModelGetter>>

Returns an IEnumerable<IGrouping<Object,IGroupingGetter>> object which can be used to read data in groups.

Remarks

Examples

GroupBy(string selectorKey, string sortKey = null) selectorKey为分组指定分组列 sortKey可以指定排序的列，如果不指定，默认使用datastore中定义的排序规则排序 以下案例演示datastore:d_department_course按studentgrade_studentid列分组 并按studentgrade_studentid列排序, 获得TModel类型数据集合

using Appeon.Data.PowerBuilder;
using System;
using System.Linq;

namespace Appeon.ApiDoc.IDataStoreExamples
{
    public class GroupByExample
    {
        private SchoolContext _context;

        public GroupByExample(SchoolContext dataContext)
        {
            // Set Data Context
            _context = dataContext;
        }

        public void Example6()
        {
            // Instantiate the datastore with datawindow:d_department_course
            var datastore = new DataStore("d_department_course", _context);

            datastore.Retrieve();

            // 设置分组列和排序列
            var studentidGropup = datastore.
                GroupBy<D_Department_Course>("studentgrade_studentid",
                                            "studentgrade_studentid");

            Console.WriteLine("Studentid Gropup Count:{0}", 
                              studentidGropup.Count());

            /*This code produces the following output:
             
            Studentid Gropup Count:26
            */
        }
    }
}
