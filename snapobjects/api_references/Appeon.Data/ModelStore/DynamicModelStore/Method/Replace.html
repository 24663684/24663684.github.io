IModelStore<TModel>.Replace(int index, TModel model) Method

.NET Standard 2.x | Current Version(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha (current)

从ModelStore中移除按索引指定的model object并插入一个新的model object。

Removes the model object specified by index from the ModelStore and insert another model object.

Namespace: Appeon.Data

Assembly: Appeon.Data.dll

Syntax

   public void Replace(int index, TModel model);

Parameters

index System.Int32

The zero-based index of the model object in the ModelStore to replace.

model TModel

The new model object which will replace the old model object.

Remarks

If the ModelStore is being tracked, the removed model object will get ModelState.Deleted state, and the new inserted model object will get ModelState.NewModified state.

Examples

用新记录替换Model中指定记录

using Appeon.Data;
using Appeon.ApiDoc.Models.School;
using System;

namespace Appeon.ApiDoc.IModelStoreExamples
{
    public class ReplaceExample
    {
        private readonly SchoolContext _context;

        public ReplaceExample(SchoolContext dataContext)
        {
            // Set data context
            _context = dataContext;
        }

        public void Example()
        {
            IModelStore<Course> courses = new ModelStore<Course>();

            // 检索出courses中的数据
            courses.Retrieve(_context);

            //设置一条新的Course记录
            Course itemCourse = new Course()
            {
                CourseID =1,
                Title = "test",
                Credits = 1,
                DepartmentID = 1
            };

            // 输出替换前Model中Index = 1 的记录的Title信息 
            Course oldCourse = courses.Get(1);
            Console.WriteLine("Title = {0}", oldCourse.Title);

            // 用newCourse替换Model中Index为1的记录
            courses.Replace(1, itemCourse);

            // 输出替换后Model中Index = 1 的记录的Title信息
            Course newCourse = courses.Get(1);
            Console.WriteLine("Title = {0}", newCourse.Title);

            /* This code example produces the following output:
            
            Title = Chemistry
            Title = test
            */
        }
    }
}
