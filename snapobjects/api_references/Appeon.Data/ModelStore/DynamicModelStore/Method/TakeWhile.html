IModelStore<TModel>.TakeWhile(Func<TModel, bool> predicate) Method

.NET Standard 2.x | Current Version(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha (current)

只要指定的条件为true，就从ModelStore获取model objects，直到一个模型对象不匹配指定的条件。

Takes model objects from the ModelStore as long as a specified condition is true until a model object doesn't match the specified condition.

Namespace: Appeon.Data

Assembly: Appeon.Data.dll

Syntax

  public IModelStore<TModel> TakeWhile(Func<TModel, bool> predicate);

Parameters

predicate System.Func<TModel,System.Boolean>

A function to test each model object for a condition.

Returns

Appeon.Data.IModelStore<TModle>

If the ModelStore is being tracked, returns a ModelStore created by clones of the model objects (before the model object at which the test no longer passes);

Otherwise, returns a ModelStore created by the model objects (before the model object at which the test no longer passes) directly.

Remarks

Examples

获取Model中指定条件的记录

using Appeon.Data;
using Appeon.ApiDoc.Models.School;
using System;

namespace Appeon.ApiDoc.IModelStoreExamples
{
    public class TakeWhileExample
    {
        private readonly SchoolContext _context;

        public TakeWhileExample(SchoolContext dataContext)
        {
            // Set data context
            _context = dataContext;
        }

        public void Example()
        {
            IModelStore<Course> courses = new ModelStore<Course>();

            // 检索出courses中的数据
            courses.Retrieve(_context);

            // 获取Model中DepartmentID ==7的记录
            var takeitem = courses.TakeWhile(m => m.DepartmentID == 7);

            // 输出执行Take后返回的记录行数
            Console.WriteLine("Take Count = {0}", takeitem.Count);

            /* This code example produces the following output:
            
            Take Count = 1
            */
        }
    }
}
