IModelStore<TModel>.FindLast(Predicate<TModel> match) Method

.NET Standard 2.x | Current Version(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha (current)

查找ModelStore中符合指定谓词的条件的model object，返回符合条件的最后一个结果。

Finds the last model object in the ModelStore which matches the conditions defined by the specified predicate.

Namespace: Appeon.Data

Assembly: Appeon.Data.dll

Syntax

  public TModel FindLast(Predicate<TModel> match);

Parameters

match System.Predicate<TModel>

A Predicate<TModel> object which specifies the the conditions.

Returns

TModel

If the current ModelStore is being tracked, returns the clone of the model object found;

Otherwise, returns the model object found directly.

Remarks

Examples

按指定条件，以倒序方式，查找Model中满足条件的数据。

using Appeon.Data;
using Appeon.ApiDoc.Models.School;
using System;

namespace Appeon.ApiDoc.IModelStoreExamples
{
    public class FindLastExample
    {
        private readonly SchoolContext _context;

        public FindLastExample(SchoolContext dataContext)
        {
            // Set data context
            _context = dataContext;
        }

        public void Example()
        {
            IModelStore<Department> dept = new ModelStore<Department>();

            //检索出Department中的数据,并进行排序
            dept.Retrieve(_context);
            dept.Sort(m => m.DepartmentID);

            //在Model中倒序查找DepartmentID = 10的记录，返回符合条件的第一条记录
            var findResult = dept.FindLast(m => m.DepartmentID == 10);

            if(findResult == null)
            {
                //如未找到匹配记录，则返回空值
                Console.WriteLine("No Find!");
            }
            else
            {
                Console.WriteLine("Department ID = {0}", findResult.DepartmentID);
            }

            /* This code example produces the following output:
            
            Department ID = 1
            */
        }
    }
}
