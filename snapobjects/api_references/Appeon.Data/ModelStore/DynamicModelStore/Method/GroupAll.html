IModelStore<TModel>.GroupAll<TKey>(Func<TModel, TKey> groupBy, bool sortByKey = false, bool decending = false) Method

从ModelStore中获取一个按指定分组规则和排序规则产生的IDictionary<TKey, IModelStore<TModel>> 对象。

Get an IDictionary<TKey, IModelStore <TModel >> object generated from the ModelStore according to the specified grouping rules and sorting rules.

Namespace: Appeon.Data

Assembly: Appeon.Data.dll

Syntax

   public IDictionary<TKey, IModelStore<TModel>> GroupAll<TKey>(Func<TModel, TKey> groupBy, bool sortByKey = false, bool decending = false);

Type Parameters

TKey

The datatype of the key in the returned dictionary.

Parameters

groupBy System.Func<TModel, TKey>

A Func<TModel, TKey> that defines the grouping rules.

sortByKey System.Boolean

True, sort by key.

False, doesn't sort by key.

Default is false.

decending System.Boolean

If sortByKey is true, the keys in the return dictionary are set to decending.

If sortByKey is false, the keys in the return dictionary is the reverse of the original order.

Default is false.

Returns

System.Collections.Generic.IDictionary<TKey, IModelStore<TModel>>

Returns an IDictionary<TKey, IModelStore <TModel >> object generated from the ModelStore according to the specified grouping rules and sorting rules.

Remarks

Examples

对Model中的数据进行分组。

using Appeon.Data;
using Appeon.ApiDoc.Models.School;
using System;

namespace Appeon.ApiDoc.IModelStoreExamples
{
    public class GroupAllExample
    {
        private readonly SchoolContext _context;

        public GroupAllExample(SchoolContext dataContext)
        {
            // Set data context
            _context = dataContext;
        }

        public void Example()
        {
            IModelStore<Department> dept = new ModelStore<Department>();

            // 检索出Department中的数据
            dept.Retrieve(_context);

            // 设置分组条件：按Name进行分组
            var modelResult = dept.GroupAll(m => m.Name);

            Console.WriteLine("Group Count = {0}", modelResult.Count);

            /* This code example produces the following output:
            
             Group Count =  4
            */
        }
    }
}
