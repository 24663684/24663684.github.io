IDataStore.GroupBy(int level, string sortKey = null) Method

.NET Standard 2.x | Current Version(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha (current)

按DataStore的指定级别的分组条件对主缓冲区的行进行分组。可以指定排序规则。

Groups rows of the primary buffer by the specified zero-based level of grouping criteria of DataStore. The sort criteria can be specified.

Namespace: Appeon.Data.PowerBuilder

Assembly: Appeon.Data.PowerBuilder.dll

Syntax

  public IEnumerable<IGrouping<object, IModelGetter>> GroupBy(int level, string sortKey = null);

Parameters

level System.Int32

The zero-based level of grouping criteria of DataStore.

sortkey System.String

A string whose value is the valid sort criteria for the DataWindow. The expression includes column names or numbers. A column number must be preceded by a pound sign (#).

Default is null which indicates it will use the current sort criteria in DataStore.

Returns

System.Collections.Generic.IEnumerable<IGrouping<System.Object, Appeon.Data.PowerBuilder.IModelGetter>>

Returns an IEnumerable<IGrouping<Object,IGroupingGetter>> object which can be used to read data in groups.

Remarks

Examples

GroupBy(int level, string sortKey = null) datastore:d_department_course包含两个分组,第一层按course_departmentid分组 第二层按course_courseid分组, 且按departmentid和course_courseid排序 GroupBy()获得分组，并返回分组所有数据集合, level为分组的层次，level从0开始表示第一层，1表示第二层...n表示n+1层 sortKey可以指定排序的列，如果不指定，默认使用datastore中定义的排序规则排序 以下案例演示GroupBy，由于course_departmentid只包含1,2,4,7这四个不同值， course_courseid包括9个不同的值，所以第一层分组有四个数据集合，第二层分组有九个数据集合

using Appeon.Data.PowerBuilder;
using System;
using System.Linq;

namespace Appeon.ApiDoc.IDataStoreExamples
{
    public class GroupByExample
    {
        private SchoolContext _context;

        public GroupByExample(SchoolContext dataContext)
        {
            // Set Data Context
            _context = dataContext;
        }

        public void Example3()
        {
            // Instantiate the datastore with datawindow:d_department_course
            var datastore = new DataStore("d_department_course", _context);

            datastore.Retrieve();

            // 获取第一层分组，共有四个数据集合
            var departmentGropup = datastore.GroupBy(0);

            Console.WriteLine("Department Gropup Count:{0}", 
                              departmentGropup.Count());

            // 获取第二层分组，共有九数据集合
            var courseGropup = datastore.GroupBy(1);
            Console.WriteLine("Course Gropup Count:{0}", courseGropup.Count());

            // 改变排序规则,使用course_title排序后获取第一层分组
            // 排序改变后，相同的的course_departmentid可能会不在一起，获取分组后的集合会变多
            departmentGropup = datastore.GroupBy(0, "course_title");

            Console.WriteLine("SortBy course_title, Department Gropup Count:{0}", 
                                departmentGropup.Count());

            /*This code produces the following output:
             
            Department Gropup Count:4
            Course Gropup Count:9
            SortBy course_title, Department Gropup Count:7
            */
        }
    }
}
