使用ContextOptions指定的隔离级别启动数据库事务。

Starts a database transaction, using the isolation level specified by the ContextOptions.

Assembly: Appeon.Data.dll

  public virtual IAdoDbTransaction BeginTransaction();

Returns

Appeon.Data.IAdoDbTransaction

Remarks

Examples

使用指定上下文开启一段事务不指定隔离层级,默认为ReadCommittedThe following code example starts a transaction using the default isolation level (ReadCommitted).

using Appeon.ApiDoc.Models.School;
using Appeon.Data;
using System.Data;

namespace Appeon.ApiDoc.DataContextExamples
{
    public class BeginTransactionExample
    {
        private readonly SchoolContext _context;
        public BeginTransactionExample(SchoolContext context)
        {
            _context = context;
        }

        public void Example1()
        {
            // Starts the transaction with the default isolation level
            _context.BeginTransaction();

            // Retrieves data in this transaction
            IModelStore<Department> dept = new ModelStore<Department>();
            dept.Retrieve(_context);
        }
    }
}
