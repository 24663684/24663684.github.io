IDataStore.IndexOf(TModel item) Method

.NET Standard 2.x | Current Version(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

确定DataStore主缓冲区中特定model实例的索引。此索引与主缓冲区中从零开始的行号相同。

Determines the index number of a specific model instance in the primary buffer of DataStore. This index is the same as the zero-based row number in the primary buffer.

Namespace: Appeon.Data.PowerBuilder

Assembly: Appeon.Data.PowerBuilder.dll

Syntax

  public int IndexOf(TModel item);

Parameters

item TModel

The model instance to locate in the primary buffer.

The type of the model must match with the current DataObject.

Returns

System.Int32

The index of the model instance if found in the primary buffer; otherwise, -1.

Remarks

Examples

IndexOf(TModel item)获取item位于datastore中的行号

The following code example demonstrates how to get the index of a row item.

using Appeon.Data.PowerBuilder;
using System;

namespace Appeon.ApiDoc.IDataStoreExamples
{
    public class IndexOfExample
    {
        private SchoolContext _context;

        public IndexOfExample(SchoolContext dataContext)
        {
            // Sets the data context
            _context = dataContext;
        }

        public void Example()
        {

            // Instantiates datastore with datawindow: d_department
            var datastore = new DataStore("d_department", _context);

            datastore.Retrieve();

            // Gets the department object of the first row
            var department = datastore.GetModel<D_Department>(0);

            // Gets the row number of department object in the datastore
            // It is the first row in datastore, so 0 should be returned
            int row = datastore.IndexOf(department);

            Console.WriteLine("Row: {0}", row);

            // Sorts datastore by departmentid desc
            // The record in the first row is moved to the fourth row
            datastore.SetSort("departmentid desc");
            datastore.Sort();

            row = datastore.IndexOf(department);

            Console.WriteLine("Row: {0}", row);

            /*This code produces the following output:

            Row: 0
            Row: 3
           */
        }
    }
}
