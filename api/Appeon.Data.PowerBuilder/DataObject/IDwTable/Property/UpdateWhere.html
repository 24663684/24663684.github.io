获取DwUpdateWhere枚举的枚举值，指示哪些列将包含在Update语句的WHERE子句中。

Gets a enumerate value of the DwUpdateWhere enumeration indicating which columns will be included in the WHERE clause of the Update statement.

命名空间： Appeon.Data.PowerBuilder

程序集： Appeon.Data.PowerBuilder.dll

语法

  DwUpdateWhere UpdateWhere { get; set; }

Property Value

Appeon.Data.PowerBuilder.DwUpdateWhere

The enumerate value of the DwUpdateWhere enumeration indicating which columns will be included in the WHERE clause of the Update statement. This value can impact performance or cause lost data when more than one user accesses the same tables at the same time.

Values are:

DwUpdateWhere.Key -- Key columns only (risk of overwriting another user's changes, but fast).

DwUpdateWhere.KeyAndUpdatableColumns -- Key columns and all updatable columns (risk of preventing valid updates; slow because SELECT statement is longer).

DwUpdateWhere.KeyAndModifiedColumns -- Key and modified columns (allows more valid updates than 1 and is faster, but not as fast as 0).

备注

Examples

获取DataStore的update规则

using Appeon.Data.PowerBuilder;
using System;

namespace Appeon.ApiDoc.IDwTableExamples
{
    public class UpdateWhereExample
    {
        private readonly SchoolContext _context;

        public UpdateWhereExample(SchoolContext context)
        {
            _context = context;
        }

        public void Example()
        {
            // 创建一个datastore对象
            IDataStore course = new DataStore("d_course", _context);

            // 获取该datastore对象使用的table
            IDwTable table = course.Object.DataWindow.Table;

            // 获取DataStore的update规则
            var express = table.UpdateWhere;

            // 输出update规则
            Console.WriteLine("Update Where = {0}", express.ToString());

            /*This code produces the following output:
            
            Update Where = KeyAndUpdatableColumns
            */
        }
    }
}
