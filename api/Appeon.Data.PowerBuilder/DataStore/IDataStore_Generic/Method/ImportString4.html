IDataStore.ImportString(string value, int startRow, int endRow, short startColumn, short endColumn, short dwStartColumn) Method

.NET Standard 2.x | Current Version(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

将来自一个字符串中按tab分割的数据插入DataStore。您可以指定在字符串中的起始行和结束行，并且可以指定要导入的字符串中第一列的编号和最后一列的编号。您还可以指定DataStore中应接收数据的第一列的编号。

Inserts the tab-separated data from a string into the DataStore. You can specify the starting and ending rows in the string, and specify the number of the first column and the number of the last column in the string to be imported. You can also specify the number of the first column in the DataStore that should receive data.

Namespace: Appeon.Data.PowerBuilder

Assembly: Appeon.Data.PowerBuilder.dll

Syntax

  public int ImportString(string value, int startRow, int endRow, short startColumn, short endColumn, short dwStartColumn);

Parameters

value System.String

A string from which you want to copy the data. The string should contain tab-separated columns.

startrow System.Int32

The zero-based number of the first detail row in the string that you want to import. If it is negative, 0 is used.

endRow System.Int32

The zero-based number of the last detail row in the string that you want to import. If it is negative, imports the rest of the rows.

startColumn System.Int16

The zero-based number of the first column in the string that you want to import. If it is negative, 0 is used.

endColumn System.Int16

The zero-based number of the last column in the string that you want to copy. If it is negative, imports the rest of the columns.

dwStartColumn System.Int16

The zero-based number of the first column in the DataStore that should receive data. The default is 0.

Returns

System.Int32

Returns the number of rows that were imported if it succeeds.

Remarks

The string must be formatted in tab-separated. Each line must end with a carriage return and a newline character (rn). If the string has four tab-separated columns, one line might look like for a tab-separated string:

col1_data\t col2_data\t col3_data\t col4_data\r\n

The string should consist of rows of data. If the data includes column headings or row labels, set the startRow and startColumn arguments to skip them. The datatypes and order of the DataStore columns must match with the columns of data in the string.

Examples

ImportString(string value, int startRow, int endRow, short startColumn, short endColumn, short dwStartColumn)) 将string startRow to endRow之间的数据导入到datastore startRow从0开始，0表示第一行，1表示第二行...,n为n+1行 startColumn，endColumn从0开始，0表示第一列，1表示第二列..,n为n+1列 endRow = -1 表示string最后一行 startColumn, endColumn指定将string从startColumn to endColumn导入到datastore中 dwStartColumn是datastore导入数据时从哪一列开始

using Appeon.Data.PowerBuilder;
using System;

namespace Appeon.ApiDoc.IDataStoreExamples
{
    public class ImportStringExample
    {
        private SchoolContext _context;

        public ImportStringExample(SchoolContext dataContext)
        {
            // Set Data Context
            _context = dataContext;
        }

        public void Example4()
        {

            // Instantiate the datastore with datawindow:d_department
            var datastore = new DataStore("d_department", _context);


            string importString = getImportString4();

            // 将string第一行到第二行间的数据导入到datastore
            // 只导入第二列到第五列间的数据到datastore
            // datastore:d_department从第二列开始导入，
            // datastore中第一列departmentid 不导入数据
            datastore.ImportString(importString, 0, 1, 1, 5, 1);

            Console.WriteLine("Rowcount:{0}", datastore.RowCount);

            for (int row = 0, rowcount = datastore.RowCount; row < rowcount; row++)
            {
                Console.WriteLine("DepartmentID:{0}; Name:{1}; Budget:{2}",
                                  datastore.GetItem<int>(row, "departmentid"),
                                  datastore.GetItem<string>(row, "name"),
                                  datastore.GetItem<decimal>(row, "budget"));
            }
            /*This code produces the following output:

            Rowcount:2
            DepartmentID:0; Name:Engineering; Budget:350000.0000
            DepartmentID:0; Name:English; Budget:120000.0000
           */
        }

        // set imported string
        private string getImportString4()
        {
            return "1\tEngineering\t350000.0000\t9/1/2007 12:00:00 AM\t2\n" +
                   "2\tEnglish\t120000.0000\t9/1/2007 12:00:00 AM\t6\n" +
                   "4\tEconomics\t200000.0000\t9/1/2007 12:00:00 AM\t4";

            /* 设置要导入的数据如下:
             
            "Departmentid"  "Name"  "Budget"        "Startdate"     "Administrator"
            1       "Engineering"   350000.0000     9 / 1 / 2007 12:00:00 AM    2
            2       "English"       120000.0000     9 / 1 / 2007 12:00:00 AM    6
            4       "Economics"     200000.0000     9 / 1 / 2007 12:00:00 AM    4
            */
        }
    }
}
