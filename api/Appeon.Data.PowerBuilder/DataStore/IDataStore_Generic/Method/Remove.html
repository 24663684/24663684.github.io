IDataStore.Remove(TModel item) Method

.NET Standard 2.x | Current Version(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

从DataStore的主缓存区中移除指定行。

Removes the specified row from the primary buffer of the DataStore.

Namespace: Appeon.Data.PowerBuilder

Assembly: Appeon.Data.PowerBuilder.dll

Syntax

  public bool Remove(TModel item);

Parameters

item TModel

The object to remove from the primary buffer of the DataStore.

TModel is the type of a model class that conforms to the current DataObject.

Returns

System.Boolean

True if the row is removed successfully.

False if no row is removed.

Remarks

The Remove method 在移除行前会调用RowDeleting method, 移除行后会调用RowDeleted method.

The Remove method will call the RowDeleting method before removing the row, and call the RowDeleted method after removing the row.

Examples

Remove(TModel item) 从datastore中移除指定item

using Appeon.Data.PowerBuilder;
using System;

namespace Appeon.ApiDoc.IDataStoreExamples
{
    public class RemoveExample
    {
        private SchoolContext _context;

        public RemoveExample(SchoolContext dataContext)
        {
            // Set Data Context
            _context = dataContext;
        }

        public void Example()
        {
            // Instantiate the datastore with datawindow:d_department
            var datastore = new DataStore("d_department", _context);

            datastore.Retrieve();

            // 获取datastore中第一行item
            var department = datastore.GetModel<D_Department>(0);

            Console.WriteLine("Before Remove, Rowcount:{0}", datastore.RowCount);

            // 移除datastore中item
            datastore.Remove(department);

            Console.WriteLine("After Remove, Rowcount:{0}", datastore.RowCount);

            /*This code produces the following output:
             
            Before Remove, Rowcount:4
            After Remove, Rowcount:3
            */
        }
    }
}
