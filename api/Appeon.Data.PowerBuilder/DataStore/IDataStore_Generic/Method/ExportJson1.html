IDataStore.ExportJson(MappingMethod mappingMethod) Method

将数据从DataStore主缓冲区导出为标准格式的JSON字符串。

Exports data from the DataStore primary buffer to a standard-format JSON string.

Namespace: Appeon.Data.PowerBuilder

Assembly: Appeon.Data.PowerBuilder.dll

Syntax

  public string ExportJson(MappingMethod mappingMethod);

Parameters

mappingMethod Appeon.Data.MappingMethod

A value of the MappingMethod enumerated datatype specifying the method for mapping columns.

Returns

System.String

Returns the JSON string.

Remarks

See JSON formats for details about the standard-format JSON.

Examples

The following code example demonstrates export department records to JSON strings using different MappingMethods. 当MappingMethod.JsonIndex时，导出json的"mapping-method"为 0; 当MappingMethod.MetaIndex，导出json的"mapping-method"为 1; 当MappingMethod.JsonKey，导出json的"mapping-method"为 2.

using Appeon.Data;
using Appeon.Data.PowerBuilder;
using System;

namespace Appeon.ApiDoc.IDataStoreExamples
{
    public class ExportJsonExample
    {
        private SchoolContext _context;

        public ExportJsonExample(SchoolContext dataContext)
        {
            // Set Data Context
            _context = dataContext;
        }

        public void Example1()
        {
            // Instantiate the datastore with datawindow object: d_department
            var datastore = new DataStore("d_department", _context);

            // Generates data in the Primary/Delete/Filter Buffers for this example.
            datastore.Retrieve();
            datastore.DeleteRow(0);
            datastore.SetFilter("departmentid < 5");
            datastore.Filter();

            // 将datastore primary buffers 数据导出为json格式, 
            // 导出的JSON中mapping -method设置为0：按Json的Index映射。
            string json1 = datastore.ExportJson(MappingMethod.JsonIndex);

            Console.WriteLine(json1);

            /*This export json file is :

            ExportJson1_JsonIndex.json  
            */

            // 将datastore primary buffers 数据导出为json格式, 
            // 导出的JSON中mapping -method设置为3：按JSON的Key映射。
            string json2 = datastore.ExportJson(MappingMethod.JsonKey);

            Console.WriteLine(json2);

            /*This export json file is :

            ExportJson1_JsonKey.json  
            */

            // 将datastore primary buffers 数据导出为json格式, 
            // 导出的JSON中mapping -method设置为2：按Meta的Index映射。
            string json3 = datastore.ExportJson(MappingMethod.MetaIndex);

            Console.WriteLine(json3);

            /*This export json file is :

            ExportJson1_MetaIndex.json  
            */
        }
    }
}
