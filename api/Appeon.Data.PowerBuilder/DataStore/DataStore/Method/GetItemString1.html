Remarks

使用GetItemString从DataStore的缓冲区获取信息。GetItemString在计算列的情况下返回格式化值，在非计算列的情况下返回未格式化的值。对于PowerBuilder DataStore，您可以使用GetItemFormattedString Method返回格式化值，或使用GetItemUnformattedString Method为任何类型的列返回未格式化的值。

Use GetItemString to get information from the DataStore's buffers. GetItemString Returnss a formatted value in the case of a computed column, and an unformatted value in the case of a noncomputed column. For PowerBuilder DataStores, you can use the GetItemFormattedString method to Returns a formatted value, or the GetItemUnformattedString method to Returns an unformatted value, for any type of column.
