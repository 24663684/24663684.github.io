Remarks

验证规则是布尔表达式，通常将列的编辑控件中的值与其他值进行比较。当用户输入的数据无法满足验证规则中建立的条件时，会发生ItemError Event。您可以在Database painter或DataStore画家中指定验证规则，并且可以使用SetValidate更改脚本中的规则。验证规则可以包括任何DataStore画家功能。如果要临时更改列的验证规则，可以使用GetValidate来获取并保存当前规则。要包含用户在验证规则中输入的值，请使用GetText Method。您可以将其返回值与验证标准进行比较。 如果验证规则包含数字，则DataStore需要美国格式的数字。在PowerBuilder中，请注意String函数使用当前系统设置格式化数字。如果使用它来构建规则，请指定生成美国符号的显示格式。
