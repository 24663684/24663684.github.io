获取指定行索引和列名位置的DateTime数据。可以指定缓冲区以及是否获取原始值。Gets data whose type is DateTime for the specified row and column from the specified buffer of the DataStore. You can obtain the data that was originally retrieved and stored in the database from the original buffer, as well as the current value in the primary, delete, or filter buffers.

Remarks

如果要从DataStore的缓冲区获取信息，请使用GetItemDateTime。要访问原始缓冲区中的行，请指定该行当前占用的缓冲区（主要，删除或过滤器）以及该缓冲区中的行数。为originalvalue指定true时，该Method从原始缓冲区获取该行的原始数据。

Use GetItemDateTime when you want to get information from the DataStore's buffers. To access a row in the original buffer, specify the buffer that the row currently occupies (primary, delete, or filter) and the number of the row in that buffer. When you specify true for originalvalue, the method gets the original data for that row from the original buffer.
