IDataStore.Evaluate<TValue>(string expression, int row) Method

.NET Standard 2.x | Current Version(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

Calculates the specified DataWindow expression using the specified row (by zero-based index number) in the primary buffer of the DataStore.

Namespace: Appeon.Data.PowerBuilder

Assembly: Appeon.Data.PowerBuilder.dll

Syntax

  public TValue Evaluate<TValue>(string expression, int row);

Type Parameters

TValue

The type of the result value.

Parameters

expression System.String

A DataWindow expression to evaluate.

row System.Int32

Returns

The result value.

Examples

The following code example demonstrates how to use the Evaluate(string expression, int row) method.

using Appeon.Data.PowerBuilder;
using System;

namespace Appeon.ApiDoc.IDataStoreExamples
{
    public class EvaluateExample
    {
        private SchoolContext _context;

        public EvaluateExample(SchoolContext dataContext)
        {
            // Sets the data context
            _context = dataContext;
        }

        public void Example2()
        {
            // Instantiates datastore with dataobject: d_department
            var datastore = new DataStore("d_department", _context);

            datastore.Retrieve();

            // This datastore has 4 records
            // Values of the first record is:
            // Name: Engineering; Budget: 350000.00

            string nameAndBudget = datastore.Evaluate<string>("name +': ' + budget)", 0);

            Console.WriteLine("NameAndBudget is {0}", nameAndBudget);

            /*This code produces the following output:
            
            NameAndBudget is Engineering: 350000.0000
            */

        }
    }
}
