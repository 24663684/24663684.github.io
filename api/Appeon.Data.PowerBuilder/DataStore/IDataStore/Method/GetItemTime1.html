IDataStore.GetItemTime(int row, short column, DwBuffer bufferType = DwBuffer.Primary, bool isOriginalValue = false) Method

.NET Standard 2.x | Current Version(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

Gets data whose type is TimeSpan for the specified row and column from the specified buffer of the DataStore. You can obtain the data that was originally retrieved and stored in the database from the original buffer, as well as the current value in the primary, delete, or filter buffers.

Namespace: Appeon.Data.PowerBuilder

Assembly: Appeon.Data.PowerBuilder.dll

Syntax

  public TimeSpan? GetItemTime(int row, short column, DwBuffer bufferType = DwBuffer.Primary, bool isOriginalValue = false);

Parameters

row System.Int32

The zero-based row number to get data.

column System.Int16

The zero-based column number to get data.

bufferType Appeon.Data.PowerBuilder.DwBuffer

The specified buffer of the DataStore.

The default is DwBuffer.Primary.

isOriginalValue System.Boolean

Whether to obtain the data that was originally retrieved.

True: obtain the data that was originally retrieved; False: obtain the current data.

The default is false.

Returns

System.TimeSpan?

Returns the TimeSpan? value in the specified row and column.

Remarks

Use GetItemTime to get information from the DataStore buffer. When you specify true for isOriginalValue, the method gets the original data for that row from the original buffer.

Examples

The following code example demonstrates how to use the GetItemTime method to get the time-type value in the specified row and column.

using Appeon.Data.PowerBuilder;
using System;

namespace Appeon.ApiDoc.IDataStoreExamples
{
    public class GetItemTimeExample
    {
        private SchoolContext _context;

        public GetItemTimeExample(SchoolContext dataContext)
        {
            // Sets the data context
            _context = dataContext;
        }

        public void Example1()
        {

            // Instantiates datastore with datawindow: d_coursedescription
            var datastore = new DataStore("d_coursedescription", _context);

            datastore.Retrieve();

            datastore.SetItem(0, "inputtime", TimeSpan.Parse("12:12:12"));

            // Gets the original value in row 1 and in column 1 and 7 of DataStore
            // Column 7 is time type
            Console.WriteLine("Original value: Description: {0};\n" +
                              "inputtime: {1}",
                          datastore.GetItem<string>(0, 1),
                          datastore.GetItemTime(0, 6, DwBuffer.Primary, true));

            // Gets the modified value in row 1 and in column 1 and 7 of DataStore
            // Column 7 is time type
            Console.WriteLine("Modified value: Description: {0};\n" +
                              "inputtime: {1}",
                          datastore.GetItem<string>(0, 1),
                          datastore.GetItemTime(0, 6, DwBuffer.Primary, false));

            /*This code produces the following output:
             
            Original value: Description: Calculus Description;
            inputtime: 08:08:08
            Modified value: Description: Calculus Description;
            inputtime: 12:12:12
            */
        }
    }
}
