IDataStore.Contains(TModel item) Method

.NET Standard 2.x | Current Version(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

Determines whether a data row exists in the primary buffer of the DataStore.

Namespace: Appeon.Data.PowerBuilder

Assembly: Appeon.Data.PowerBuilder.dll

Syntax

  public bool Contains(TModel item);

Parameters

item System.TModel

A TModel object which represents a row of data.

Returns

System.Boolean

True if the data row is found in the primary buffer of the DataStore; otherwise, false.

Remarks

This method compares the reference address, not the data values. It returns true when the reference address of the TModel object is the same that of a DataStore row (even if the data values are different). See examples for elaboration.

Examples

The following code example demonstrates how to use the Contains method.

using Appeon.Data.PowerBuilder;
using System;

namespace Appeon.ApiDoc.IDataStoreExamples
{
    public class ContainsExample
    {
        private SchoolContext _context;

        public ContainsExample(SchoolContext dataContext)
        {
            // Sets the data context
            _context = dataContext;
        }

        public void Example()
        {


            // Instantiates datastore with datawindow: d_department
            var datastore = new DataStore("d_department", _context);

            datastore.Retrieve();

            // Creates a department TModel object and populates it with data
            var department = new D_Department();
            department.Departmentid = 10;
            department.Name = "department name";
            department.Budget = 10m;
            department.Startdate = DateTime.Now;
            department.Administrator = 2;

            // Contains method returns false
            Console.WriteLine("DataStore contains item? {0}",
                datastore.Contains(department));

            // Appends the department data to datastore
            datastore.Add(department);

            // Contains method returns true
            Console.WriteLine("DataStore contains item? {0}",
                datastore.Contains(department));

            // Changes a data value
            department.Administrator = 3;

            // Contains method still returns true 
            // because it compares the reference address, not values
            Console.WriteLine("Datastore contains item? {0}",
                datastore.Contains(department));

            /*This code produces the following output:
            
            DataStore contains item? False
            DataStore contains item? True
            DataStore contains item? True     
            */

        }
    }
}
