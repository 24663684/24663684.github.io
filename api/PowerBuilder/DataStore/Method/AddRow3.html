IDataStore.AddRow<TModel>(IEnumable<TModel> list) Method

.NET Standard 2.x | Current Version(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

将一个TModel类型的集合添加到DataStore主缓冲区的末尾并返回起始位置的从零开始的索引。TModel必须与当前DataStore的DataObject属性匹配。

Generic method. Appends a collection of TModel instances as row(s) to the end of the primary buffer in the DataStore and returns the zero-based index of the first row added. TModel must match with the DataObject property of the DataStore.

Namespace: Appeon.Data.PowerBuilder

Assembly: Appeon.Data.PowerBuilder.dll

Syntax

  public int AddRow<TModel>(IEnumerable<TModel> list);

Type Parameters

TModel

The type of a model class that matches with the current DataObject.

Parameters

list System.Collections.Generic.IEnumerable<TModel>

一个与当前DataStore中的DataObject属性匹配的TModel实例的集合。

A collection of TModel instances which match with the DataObject property of the DataStore.

Returns

System.Int32

Returns the zero-based index number of the first row added.

Remarks

The status flag for a newly added row is ModelState.NewModified, and the row is included in the modified count.

Examples

演示通过AddRow方法在datastore中增加多行数据

The following code example demonstrates how to use the AddRow method to add multiple rows of data to the DataStore.

using Appeon.Data.PowerBuilder;
using System;
using System.Collections.Generic;

namespace Appeon.ApiDoc.IDataStoreExamples
{
    public class AddRowExample
    {
        private SchoolContext _context;

        public AddRowExample(SchoolContext dataContext)
        {
            // Sets the data context
            _context = dataContext;
        }

        public void Example3()
        {
            // Instantiates datastore with dataobject: d_department
            var datastore = new DataStore("d_department", _context);

            Console.WriteLine("Before AddRow, Rowcount: {0}", datastore.Retrieve());

            // Creates a list of department TModel objects
            var departmentList = new List<D_Department>();

            var department = new D_Department();
            department.Departmentid = 10;
            department.Name = "department name";
            department.Budget = 10;
            department.Startdate = DateTime.Now;
            department.Administrator = 1;

            departmentList.Add(department);

            var department2 = new D_Department();
            department2.Departmentid = 20;
            department2.Name = "department name2";
            department2.Budget = 20;
            department2.Startdate = DateTime.Now;
            department2.Administrator = 2;

            departmentList.Add(department2);

            // Adds the model list to datastore
            // AddRow returns a value that is the number of the last row plus 1.
            // Lower bound of C# array is 0, but that of PB array is 1, so the
            // number of current row returned from C# is 1 less than that from PB.
            int currentRow = datastore.AddRow< D_Department>(departmentList);

            Console.WriteLine("Addrow return value: {0}", currentRow);

            // Commits changes to database
            datastore.Update();

            Console.WriteLine("After Update and Retrieve, Rowcount: {0}", 
                datastore.Retrieve());

            /*This code produces the following output:
            
            Before AddRow, Rowcount: 4
            Addrow return value: 4
            After Update and Retrieve, Rowcount: 6
            */
        }
    }
}
