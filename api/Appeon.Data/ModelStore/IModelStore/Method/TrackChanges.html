改变ModelStore的追踪策略。

Changes the tracking strategy of ModelStore.

Namespace: Appeon.Data

Assembly: Appeon.Data.dll

Syntax

  public IModelStore<TModel> TrackChanges(ChangeTrackingStrategy changeTrackingStrategy);

Parameters

changeTrackingStrategy Appeon.Data.ChangeTrackingStrategy

A value of the ChangeTrackingStrategy enumerated datatype.

Returns

Appeon.Data.IModelStore<TModel>

Returns the current ModelStore which has cleared the original tracked information and began to use the new tracking strategy.

Remarks

If the TrackChanges method is used to change the tracking strategy to ChangeTrackingStrategy.Snapshot or ChangeTrackingStrategy.PropertyState, the existing references to the internal model object hold outside of ModelStore (for example, you got a reference of an internal model object by calling GetInternalModel method) cannot be used any more.

Examples

The following code example demonstrates how to set the tracking strategy of a ModelStore.设置ModelStore的ChangeTrackingStrategy

using Appeon.Data;
using Appeon.ApiDoc.Models.School;
using System;

namespace Appeon.ApiDoc.IModelStoreExamples
{
    public class TrackChangesExample
    {
        private readonly SchoolContext _context;

        public TrackChangesExample(SchoolContext dataContext)
        {
            // Sets the data context
            _context = dataContext;
        }

        public void Example()
        {
            IModelStore<Course> courses = new ModelStore<Course>();

            // Retrieves data from courses
            courses.Retrieve(_context);

            // Sets the ModelStore ChangeTrackingStrategy = PropertyState
            courses.TrackChanges(ChangeTrackingStrategy.PropertyState);

            // Sets the value for column 2 in row 1
            courses.SetValue(0, 1, "test");

            // Saves ModelStore
            var dbResult = courses.SaveChanges(_context);

            Console.WriteLine("{0} records have been updated into the database.",
                              dbResult.AffectedCount);

            /* This code example produces the following output:
            
            1 records have been updated into the database.
            */
        }
    }
}
