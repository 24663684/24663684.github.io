IModelStore<TModel>.GroupAll<TKey>(Func<TModel, TKey> groupBy, bool sortByKey = false, bool descending = false) Method

.NET Standard 2.x | Current Version (0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha (current)

Groups the data in ModelStore according to the specified grouping rules and sorting rules, and returns an IDictionary<TKey, IModelStore <TModel >> object which contains the grouped data.

Namespace: Appeon.Data

Assembly: Appeon.Data.dll

Syntax

   public IDictionary<TKey, IModelStore<TModel>> GroupAll<TKey>(Func<TModel, TKey> groupBy, bool sortByKey = false, bool descending = false);

Type Parameters

TKey

The datatype of the key in the returned IDictionary object.

Parameters

groupBy System.Func<TModel, TKey>

A Func<TModel, TKey> that defines the grouping rules.

sortByKey System.Boolean

True: to sort by key.

False (default): not to sort by key.

descending System.Boolean

True: when sortByKey is true, the keys in the return IDictionary object are set to descending; when sortByKey is false, the keys in the return IDictionary object are in the reverse of the original order.

False (default): when sortByKey is true, the keys in the return dictionary are set to ascending; when sortByKey is false, the keys in the return dictionary are in the original order.

Returns

System.Collections.Generic.IDictionary<TKey, IModelStore<TModel>>

Returns an IDictionary<TKey, IModelStore <TModel >> object which contains the data grouped and sorted by the specified rules.

Examples

The following code example demonstrates how to group the data in a ModelStore.

using Appeon.Data;
using Appeon.ApiDoc.Models.School;
using System;

namespace Appeon.ApiDoc.IModelStoreExamples
{
    public class GroupAllExample
    {
        private readonly SchoolContext _context;

        public GroupAllExample(SchoolContext dataContext)
        {
            // Sets the data context
            _context = dataContext;
        }

        public void Example()
        {
            var studentGrades = new ModelStore<StudentGrade>();

            // Retrieves data.
            studentGrades.Retrieve(_context);

            // Groups by CourseID.
            var groupResult = studentGrades.GroupAll(m => m.CourseID);

            // Shows the grouped data.
            Console.WriteLine("Group Count = {0}", groupResult.Count);
            foreach (var group in groupResult)
            {
                Console.WriteLine("CourseID: {0}", group.Key);

                // Gets a ModelStore object created by model objects which has the 
                // same course ID.
                var ms = group.Value;
                foreach (var studentGrade in ms)
                {
                    Console.WriteLine("Student ID: {0} Grade: {1}",
                        studentGrade.StudentID,
                        studentGrade.Grade);
                }

                // Only shows the first group for this example.
                break;
            }

            /* This code example produces the following output:
            
            Group Count = 9
            CourseID: 2021
            Student ID: 2 Grade: 4.00
            Student ID: 3 Grade: 3.00
            Student ID: 6 Grade: 2.50
            Student ID: 7 Grade: 3.50
            Student ID: 8 Grade: 3.00
            */
        }
    }
}
