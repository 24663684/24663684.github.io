IModelStore<TModel>.ExportPlainJson(DwBuffer dwbuffer, int startRow, int endRow, short startColumn, short endColumn) Method

.NET Standard 2.x | Current Version (0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha (current)

Exports the model objects (from the specified range) and the properties (from the specified range) from ModelStore to a simple-format JSON string.

Namespace: Appeon.Data

Assembly: Appeon.Data.dll

Syntax

  public string ExportPlainJson(DwBuffer dwbuffer, int startRow, int endRow, short startColumn, short endColumn);

Parameters

startRow System.Int32

The zero-based number of the first model object in ModelStore that you want to export.

endRow System.Int32

The zero-based number of the last model object in ModelStore that you want to export. If it is negative, it indicates the rest of model objects.

startColumn System.Int16

The zero-based index of the first property in the model object that you want to export. If it is negative, 0 is used.

endColumn System.Int16

The zero-based index of the last property in the model object that you want to export. If it is negative, it indicates the rest of properties.

Returns

System.String

Returns the JSON string if it succeeds.

Examples

The following code example demonstrates how to use the ExportPlainJson(Int32, Int32, Int16, Int16) method to export data to a JSON string using startRow, endRow, startColumn, and endColumn parameters.

using Appeon.ApiDoc.Models.School;
using Appeon.Data;
using System;

namespace Appeon.ApiDoc.IModelStoreExamples
{
    public class ExportPlainJsonExample
    {
        private SchoolContext _context;

        public ExportPlainJsonExample(SchoolContext dataContext)
        {
            // Sets the data context
            _context = dataContext;
        }

        public void Example6()
        {
            // Creates ModelStore and retrieves data.
            var department =
                new ModelStore<Department>(ChangeTrackingStrategy.PropertyState);
            department.Retrieve(_context);

            // Exports model objects at index 0 through the end,
            // and properties at index 1 through 3.
            string json1 =
                department.ExportPlainJson(0, -1, 1, 3);

            Console.WriteLine(json1);

            /* The exported JSON file is:

            ExportPlainJson6.json
            */
        }
    }
}
