IModelStore<TModel>.Find(Predicate<TModel> match) Method

.NET Standard 2.x | Current Version (0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha (current)

Searches for the model object which meets the condition defined by the predicate and returns the first occurrence.

Namespace: Appeon.Data

Assembly: Appeon.Data.dll

Syntax

  public TModel Find(Predicate<TModel> match);

Parameters

match System.Predicate<TModel>

A Predicate<TModel> object which specifies the condition.

Returns

TModel

If the current ModelStore is being tracked, returns the copy of the model object found; otherwise, returns the model object found directly.

Remarks

Examples

The following code example demonstrates how to find the model object that meets the specified criteria and return the first model object found.

using Appeon.Data;
using Appeon.ApiDoc.Models.School;
using System;

namespace Appeon.ApiDoc.IModelStoreExamples
{
    public class FindExample
    {
        private readonly SchoolContext _context;

        public FindExample(SchoolContext dataContext)
        {
            // Sets the data context
            _context = dataContext;
        }

        public void Example()
        {
            var dept = new ModelStore<Department>();

            // Retrieves data from Department and sorts data
            dept.Retrieve(_context);
            dept.Sort(m => m.DepartmentID);

            // Finds the model object whose DepartmentID is 1 and returns the first 
            // model object found.
            var findResult = dept.Find(m => m.DepartmentID == 1);

            if (findResult == null)
            {
                // If no model object found, returns an empty string.
                Console.WriteLine("Not Found!");
            }
            else
            {
                Console.WriteLine("Department ID: {0}", findResult.DepartmentID);
            }

            /* This code example produces the following output:
            
            Department ID: 1
            */
        }
    }
}
