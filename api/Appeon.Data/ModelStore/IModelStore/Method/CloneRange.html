IModelStore<TModel>.CloneRange(int index, int count) Method

获取由零基础索引start和count指定的model objects的副本的collection。

Copies a number of model objects starting from the specified position.

Namespace: Appeon.Data

Assembly: Appeon.Data.dll

Syntax

  public IList<TModel> CloneRange(int index, int count);

Parameters

index System.Int32

The zero-based index from which to start copying.

count System.Int32

A number specifying how many model objects will be copied.

Returns

System.Collections.Generic.IList<TModel>

A new collection of model objects.

备注

Examples

The following code example demonstrates how to Clone ModelStore中指定的多行数据。

using Appeon.Data;
using Appeon.ApiDoc.Models.School;

namespace Appeon.ApiDoc.IModelStoreExamples
{
    public class CloneRangeExample
    {
        private readonly SchoolContext _context;

        public CloneRangeExample(SchoolContext dataContext)
        {
            // Sets the data context
            _context = dataContext;
        }

        public void Example()
        {
            IModelStore<Department> dept = new ModelStore<Department>();

            // Retrieves data from Department
            dept.Retrieve(_context);

            //Clone ModelStore中从index = 1 行开始的两行数据
            dept.CloneRange(1, 2);

        }
    }
}
