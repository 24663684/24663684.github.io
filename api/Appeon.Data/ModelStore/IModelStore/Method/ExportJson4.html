IModelStore<TModel>.ExportJson(int startRow, int endRow, MappingMethod mappingMethod) Method

.NET Standard 2.x | Current Version (0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha (current)

Exports the model objects (from the specified range) from ModelStore to a standard-format JSON string.

Namespace: Appeon.Data

Assembly: Appeon.Data.dll

Syntax

  public string ExportJson(int startRow, int endRow, MappingMethod mappingMethod);

Parameters

startRow System.Int32

The zero-based index of the first model object in ModelStore that you want to export.

endRow System.Int32

The zero-based index of the last model object in ModelStore that you want to export. If it is negative, it indicates the rest of model objects.

mappingMethod Appeon.Data.MappingMethod

A value of the MappingMethod enumerated datatype specifying the method for mapping columns.

  • MappingMethod.JsonIndex -- Use the index of JSON item to map with the DataStore column. The "mapping-method" value in the exported JSON string is 0.
  • MappingMethod.MetaIndex -- Use the index of meta-columns to map with the DataStore column. The "mapping-method" value in the exported JSON string is 1.
  • MappingMethod.JsonKey -- Use the key of JSON item to map with the DataStore column. The "mapping-method" value in the exported JSON string is 2.

Returns

System.String

Returns the JSON string if it succeeds.

Examples

The following code example demonstrates how to use the ExportJson(Int32, Int32, MappingMethod) method to export two department records (starting at index 0 and ending at index 1) to a JSON string.

using Appeon.ApiDoc.Models.School;
using Appeon.Data;
using System;

namespace Appeon.ApiDoc.IModelStoreExamples
{
    public class ExportJsonExample
    {
        private SchoolContext _context;

        public ExportJsonExample(SchoolContext dataContext)
        {
            // Sets the data context
            _context = dataContext;
        }

        public void Example4()
        {
            // Creates ModelStore and retrieves data. 
            var department = new ModelStore<Department>();
            department.Retrieve(_context);

            // Exports model object (starts at index 0 and ends at index 1). 
            string json = department.ExportJson(0, 1, MappingMethod.JsonIndex);

            Console.WriteLine(json);

            /* The exported JSON file is:

            ExportJson4.json  
            */
        }
    }
}
