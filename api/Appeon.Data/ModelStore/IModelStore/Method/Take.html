IModelStore<TModel>.Take(int count) Method

.NET Standard 2.x | Current Version (0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

从ModelStore的开头返回由指定数量的连续model objects创建的新ModelStore。

Returns the specified number of contiguous model objects in a new ModelStore from the start of the current ModelStore.

Namespace: Appeon.Data

Assembly: Appeon.Data.dll

Syntax

  public IModelStore<TModel> Take(int count);

Parameters

count System.Int32

The number of model objects to take.

Returns

Appeon.Data.IModelStore<TModle>

If ModelStore is being tracked, returns the copies of the model objects; otherwise, returns the model objects directly.

Remarks

Examples

获取ModelStore中多行记录

The following code example demonstrates how to take the first two records from ModelStore.

using Appeon.Data;
using Appeon.ApiDoc.Models.School;
using System;

namespace Appeon.ApiDoc.IModelStoreExamples
{
    public class TakeExample
    {
        private readonly SchoolContext _context;

        public TakeExample(SchoolContext dataContext)
        {
            // Sets the data context
            _context = dataContext;
        }

        public void Example()
        {
            IModelStore<Course> courses = new ModelStore<Course>();

            // Retrieves data from courses
            courses.Retrieve(_context);

            // Gets the first two records
            var takeitem = courses.Take(2);

            // Shows the number of records that are taken
            Console.WriteLine("Take Count = {0}", takeitem.Count);

            /* This code example produces the following output:
            
            Take Count = 2
            */
        }
    }
}
