IModelStore<TModel>.ExportPlainJson(int startRow) Method

.NET Standard 2.x | Current Version (0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

将model objects（从指定的起始位置到结尾）从ModelStore导出为简单格式的JSON字符串。

Exports the model objects (from the specified starting position to the end) in ModelStore to a simple-format JSON string.

Namespace: Appeon.Data

Assembly: Appeon.Data.dll

Syntax

  public string ExportJson(int startRow);

Parameters

startRow System.Int32

The zero-based index of the first model object you want to export.

Returns

System.String

Returns the JSON string if it succeeds.

Remarks

See JSON formats for details about the simple-format JSON.

Examples

The following code example demonstrates how to export the department records in rows 2 through the end.

using Appeon.ApiDoc.Models.School;
using Appeon.Data;
using System;

namespace Appeon.ApiDoc.IModelStoreExamples
{
    public class ExportPlainJsonExample
    {
        private SchoolContext _context;

        public ExportPlainJsonExample(SchoolContext dataContext)
        {
            // Sets the data context
            _context = dataContext;
        }

        public void Example3()
        {
            // Creates ModelStore and retrieves data 
            IModelStore<Department> department = new ModelStore<Department>();
            department.Retrieve(_context);

            // Exports records starting from rows 2 through the end 
            string json = department.ExportPlainJson(1);

            Console.WriteLine(json);

            /* The exported JSON file is:

            ExportPlainJson3.json  
            */
        }
    }
}
