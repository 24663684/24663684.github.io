IModelStore<TModel>.CopyTo(int index, TModel[] array, int arrayIndex, int count) Method

.NET Standard 2.x | Current Version (0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

将指定数量的从指定的ModelStore索引开始的model objects复制到一个数组。

Copies a number of the model objects (starting from the specified position) of ModelStore to the specified position of an array.

Namespace: Appeon.Data

Assembly: Appeon.Data.dll

Syntax

  public void CopyTo(int index, TModel[] array, int arrayIndex, int count);

index System.Int32

The index of a data item in ModelStore at which copying begins.

array TModel[]

The one-dimensional System.Array to which the model objects of ModelStore will be copied.

arrayIndex System.Int32

The zero-based index of the array at which copying begins.

count System.Int32

A number specifying how many data items will be copied.

Remarks

Examples

将ModelStore中的数据copy到model数组中。指定目标数组的插入位置索引。指定要copy的ModelStore中的记录索引，及要拷贝的记录数

using Appeon.Data;
using Appeon.ApiDoc.Models.School;
using System;

namespace Appeon.ApiDoc.IModelStoreExamples
{
    public class CopyToExample
    {
        private readonly SchoolContext _context;

        public CopyToExample(SchoolContext dataContext)
        {
            // Sets the data context
            _context = dataContext;
        }

        public void Example3()
        {
            // 创建ModelStore对象，并检索数据
            IModelStore<Course> currentCourse = new ModelStore<Course>();
            currentCourse.Retrieve(_context);

            // 创建一个Course数组
            Course[] courses = new Course[5];

            // 指定从数组的index = 2开始，将ModelStore中从index=2位置开始的两条数据拷贝到该数组中
            currentCourse.CopyTo(2, courses, 2, 2);

            // 循环输出数组中的数据
            foreach (Course m in courses)
            {
                if (m == null)
                {
                    Console.WriteLine("null");
                }
                else
                {
                    Console.WriteLine(m.Title);
                }

            }

            /*This code produces the following output:
             
            null
            null
            Physics
            Composition
            null
            */

        }
    }
}
