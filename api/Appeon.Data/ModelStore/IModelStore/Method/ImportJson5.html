IModelStore<TModel>.ImportJson(string json, int startRow, int endRow, short startProperty, short endProperty) Method

.NET Standard 2.x | Current Version (0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha (current)

Inserts data from a JSON string into ModelStore. You can specify the starting and ending positions in the JSON array, and you can also specify the number of the first key and the last key in the JSON object to be imported. Data that are out of this range will not be imported into ModelStore.

Namespace: Appeon.Data

Assembly: Appeon.Data.dll

Syntax

  public int ImportJson(string json, int startRow, int endRow, short startProperty, short endProperty);

Parameters

json System.String

A string specifying the JSON data.

The JSON string must comply with the simple-format JSON or standard-format JSON.

startRow System.Int32

The zero-based number of the first detail object in the JSON array that you want to import. If it is negative, 0 is used.

endRow System.Int32

The zero-based number of the last detail object in the JSON array that you want to import. If it is negative, it indicates the rest of rows.

startProperty System.Int32

The zero-based number of the first key in the JSON object that you want to import. If it is negative, 0 is used.

endProperty System.Int32

The zero-based number of the last key in the JSON object that you want to import. If it is negative, it indicates the rest of columns.

Returns

System.Int32

Returns the number of model objects that were imported if it succeeds.

Examples

The following code example demonstrates how to use the ImportJson(String, Int32, Int32, Int16, Int16) method to import the department data from between the starting detail object and the ending detail object and from between the starting key and the ending key in a JSON string to ModelStore.

using Appeon.ApiDoc.Models.School;
using Appeon.Data;
using System;
using System.IO;

namespace Appeon.ApiDoc.IModelStoreExamples
{
    public class ImportJsonExample
    {
        private SchoolContext _context;

        public ImportJsonExample(SchoolContext dataContext)
        {
            // Sets the data context
            _context = dataContext;
        }

        public void Example5()
        {
            // Creates ModelStore
            var department = new ModelStore<Department>();

            // Gets the JSON file. ImportJson2.json contains 4 records.
            string path = Path.Combine(AppContext.BaseDirectory,
                                       @"Jsons\IModelStore\ImportJson2.json");
            string json = File.ReadAllText(path);

            // Imports data from the first detail object through the end,
            // and from the first key to the fifth key.
            department.ImportJson(json, 0, 3, 0, 4);

            Console.WriteLine("Count: {0}", department.Count);

            /* This code produces the following output:

            Count: 4
           */
        }
    }
}
