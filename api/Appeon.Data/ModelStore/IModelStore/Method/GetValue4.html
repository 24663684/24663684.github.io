IModelStore<TModel>.GetValue(int index, string propertyName, bool bOriginalValue = false) Method

.NET Standard 2.x | Current Version (0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha (current)

Gets the value of the specified property (by name) of the specified model object (by index) in ModelStore. It gets the current value by default.

Namespace: Appeon.Data

Assembly: Appeon.Data.dll

Syntax

  public object GetValue(int index, string propertyName, bool bOriginalValue = false);

Parameters

index System.Int32

The zero-based index of the model object in ModelStore.

propertyName System.String

The property name in the model object from which to get value.

bOriginalValue System.Boolean

True: to get the original value if ModelStore is being tracked.

False (default): to get the current value.

Default is false.

Returns

System.Object

Returns the value in the specified model object and property. The datatype of the returned data is object.

Examples

The following code example demonstrates how to get the value of the specified row and column (by name). The data type is not specified.

using Appeon.Data;
using Appeon.ApiDoc.Models.School;
using System;

namespace Appeon.ApiDoc.IModelStoreExamples
{
    public class GetValueExample
    {
        private readonly SchoolContext _context;

        public GetValueExample(SchoolContext dataContext)
        {
            // Sets the data context
            _context = dataContext;
        }

        public void Example4()
        {
            var depts = new ModelStore<Department>();

            // Retrieves data.
            depts.Retrieve(_context);

            // Gets the value of the "Name" property of the model object at index 0.
            var name = depts.GetValue(0, "Name");

            Console.WriteLine("Department Name: {0}", name);


            // Creates a ModelStore and sets the tracking strategy to PropertyState.
            var depts2 =
                new ModelStore<Department>(ChangeTrackingStrategy.PropertyState);

            // Retrieves data.
            depts2.Retrieve(_context);

            // Modifies the value.
            depts2.SetValue(0, "Name", "New Department");

            // Gets the original value.
            name = depts2.GetValue(0, "Name", true);
            Console.WriteLine();
            Console.WriteLine("Department Name (Orginal): {0}", name);

            // Gets the current value.
            name = depts2.GetValue(0, "Name");
            Console.WriteLine("Department Name (Current): {0}", name);

            /* This code example produces the following output:
            
            Department Name: Engineering

            Department Name (Orginal): Engineering
            Department Name (Current): New Department
            */
        }
    }
}
