IModelStore<TModel>.Filter(Func<TModel, bool> predicate, bool removeFiltered = false) Method

.NET Standard 2.x | Current Version (0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

获取一个新的ModelStore,它由当前ModelStore中过滤条件匹配的model objects创建, 可以选择是否将满足过滤的model objects从当前ModelStore中移除。

Filters the model objects which meets the filter criteria and returns a new ModelStore containing the filtered model objects. You can choose whether to remove the filtered model objects from the current ModelStore.

Namespace: Appeon.Data

Assembly: Appeon.Data.dll

Syntax

  public IModelStore<TModel> Filter(Func<TModel, bool> predicate, bool removeFiltered = false);

Parameters

predicate System.Func<TModel, System.Boolean>

A Func<TModel, bool> which specifies the filter criteria.

removeFiltered System.Boolean

True: to remove the filtered model objects from the current ModelStore.

False (default): not to remove the filtered model objects from the current ModelStore.

Returns

Appeon.Data.IModelStore<TModel>

If the current ModelStore is being tracked, and removeFiltered is false, returns a new ModelStore which contains the copies of the filtered model objects; otherwise, returns a new ModelStore which contains the model objects that are filtered and moved from the current ModelStore.

Remarks

Examples

The following code example demonstrates how to use the Filter method to filter data by the specified criteria.按指定条件，过滤出ModelStore中的数据。

using Appeon.Data;
using Appeon.ApiDoc.Models.School;
using System;

namespace Appeon.ApiDoc.IModelStoreExamples
{
    public class FilterExample
    {
        private readonly SchoolContext _context;

        public FilterExample(SchoolContext dataContext)
        {
            // Sets the data context
            _context = dataContext;
        }

        public void Example()
        {
            IModelStore<Department> dept = new ModelStore<Department>();

            //  Retrieves data from Department
            dept.Retrieve(_context);

            // Filters the records whose DepartmentID = 2
            // Sets removeFiltered = false, so after filter, 
            // the records remain the same in the original ModelStore
            var filterModel = dept.Filter(m => m.DepartmentID == 2, false);

            Console.WriteLine("removeFiltered = false");
            Console.WriteLine("Filter Count = {0}", filterModel.Count);
            Console.WriteLine("Total Count = {0}", dept.Count);

            /* This code example produces the following output:
            
            removeFiltered = false
            Filter Count = 1
            Total Count = 4
            */

            // Sets removeFiltered = false, so after filter, 
            // the filtered records are removed from the original ModelStore
            var newfilterModel = dept.Filter(m => m.DepartmentID == 2, true);

            Console.WriteLine("removeFiltered = true");
            Console.WriteLine("Filter Count = {0}", newfilterModel.Count);
            Console.WriteLine("Total Count = {0}", dept.Count);

            /* This code example produces the following output:
            
            removeFiltered = true
            Filter Count = 1
            Total Count = 3
            */
        }
    }
}
