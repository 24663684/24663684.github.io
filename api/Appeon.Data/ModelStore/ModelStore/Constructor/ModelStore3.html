使用一个IEnumerable<IModelEntry<TModel>>对象初始化一个ModelMapper<TModel>类的新实例。 可指定数据变化追踪策略，默认不追踪数据变化。

Initializes a new instance of the ModelStore class by an IEnumerable<IModel<TModel>> object. You can specify the change tracking strategy. Data changes are not tracked by default.

命名空间： Appeon.Data

程序集： Appeon.Data.dll

语法

  public ModelStore(IEnumerable<TModel> models, ChangeTrackingStrategy strategy = ChangeTrackingStrategy.NotTrack);

Parameters

models System.Collections.Generic.IEnumerable<TModel>

A sequence of TModel objects which contain data.

strategy Appeon.Data.ChangeTrackingStrategy

(Optional) A value of the ChangeTrackingStrategy enumerated datatype. The default is ChangeTrackingStrategy.NotTrack.

Examples

实例化一个ModelStore对象, 并指定跟踪策略实例化对象时，为对象传入Model数据The following code example initializes a ModelStore object, specifies the tracking strategy, and inserts the model object to ModelStore.

using Appeon.ApiDoc.Models.School;
using Appeon.Data;
using System;
using System.Collections.Generic;

namespace Appeon.ApiDoc.ModelStoreExamples
{
    public class ModelStoreExample
    {
        private SchoolContext _context;

        public ModelStoreExample(SchoolContext dataContext)
        {
            // Sets the data context
            _context = dataContext;
        }

        public void Example3()
        {
            // Creates two Course records
            IList<Course> courseRange = new List<Course>();
            Course course = new Course()
            {
                CourseID = 1,
                Title = "test one",
                Credits = 1,
                DepartmentID = 1
            };

            Course course2 = new Course()
            {
                CourseID = 2,
                Title = "test two",
                Credits = 1,
                DepartmentID = 1
            };

            // Adds two Course records to the list
            courseRange.Add(course);
            courseRange.Add(course2);

            // Initializes a ModelStore object
            // Specifies the tracking strategy to PropertyState
            // Inserts Course records to ModelStore
            IModelStore<Course> model = new ModelStore<Course>(courseRange, 
                                ChangeTrackingStrategy.PropertyState);

            // Shows how many rows in ModelStore
            Console.WriteLine("Item Count = {0}.",
                model.Count);

            /*This code produces the following output:

            Item Count = 2.
           */
        }
    }
}
