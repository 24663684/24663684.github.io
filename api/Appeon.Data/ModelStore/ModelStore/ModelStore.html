0.5.0-alpha 0.5.1-alpha(current)
ModelStore是一个基于.NET Standard开发的数据操作组件，基于Model元数据（Model Attribute）构建数据存储结构和数据操作行为，并且提供了DynamicModel支持，可以使用SQL Executor执行SQL查询获取对应的ModelStore对象，提供数据状态跟踪、遍历、统计等功能，与DataStore相比提供了更轻量化的数据操作特性。

The C# ModelStore is a data manipulation component that can be used in place of the C# DataStore. It constructs data storage structures and defines data operations using Model Attributes. And it can be created and maintained using any C# IDE or even a simple text editor.

Just like the C# DataStore, you can create ModelStore objects without writing any SQL code. ModelStore objects are created through annotations by simply specifying the attributes of its model. The necessary SQL statements appropriate for the target database will be automatically generated and validated by the C# framework.

The C# ModelStore provides most but not alsl the capabilities of the C# DataStore. For example, you can then use ModelStore objects for common data operations such as data caching, filtering, sorting, status tracking, data traversing, and statistical analysis.
