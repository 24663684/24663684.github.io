IModelStore<TModel>.FindAll(Predicate<TModel> match); Method

.NET Standard 2.x | Current Version(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

查找ModelStore中符合指定谓词的条件的所有model objects, 返回一个由这些找到的model objects创建的新的ModelStore。

Finds the all model objects in the ModelStore which matches the conditions defined by the specified predicate and returns a new ModelStore created by the model objects found.

Namespace: Appeon.Data

Assembly: Appeon.Data.dll

Syntax

  public IModelStore<TModel> FindAll(Predicate<TModel> match);

Parameters

match System.Predicate<TModel>

A Predicate<TModel> object which specifies the conditions.

Returns

Appeon.Data.IModelStore<TModel>

If the current ModelStore is being tracked, returns a new ModelStore created by the clones of the model objects found;

Otherwise, returns a new ModelStore created by the model objects found directly.

Remarks

Examples

按指定条件，查找Model中满足条件的记录。

using Appeon.Data;
using Appeon.ApiDoc.Models.School;
using System;

namespace Appeon.ApiDoc.IModelStoreExamples
{
    public class FindAllExample
    {
        private readonly SchoolContext _context;

        public FindAllExample(SchoolContext dataContext)
        {
            // Set data context
            _context = dataContext;
        }

        public void Example()
        {
            IModelStore<Department> dept = new ModelStore<Department>();

            //检索出Department中的数据
            dept.Retrieve(_context);

            //在Model中查找全部DepartmentID = 10的记录
            var findResult = dept.FindAll(m => m.DepartmentID == 10);

            //如未查找到符合的记录，则findResult.Count = 0
            Console.WriteLine("Find Count = {0}", findResult.Count);

            /* This code example produces the following output:
            
            Find Count = 1
            */
        }
    }
}
