IModelStore<TModel>.SetPropertyState(int index, string propertyName, PropertyState propertyState) Method

.NET Standard 2.x | Current Version(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

当ModelStore正在被追踪，设置ModelStore中的一个model object(按基于零的索引指定)的一个属性(按property name指定)的状态.

When the ModelStore is being tracked, sets the state of a property (specified by the property name) of the model object (specified by a zero-based index) in the ModelStore .

Namespace: Appeon.Data

Assembly: Appeon.Data.dll

Syntax

  public void SetPropertyState(int index, string propertyName, PropertyState propertyState);

Parameters

index System.Int32

The zero-based index of the model object in the ModelStore.

propertyName System.String

The property name of the model object to set property state.

propertyState Appeon.Data.PropertyState

A value of the PropertyState enumerated datatype.

Remarks

Examples

设置Model中指定记录中指定属性的状态 以属性名称进行设置

using Appeon.Data;
using Appeon.ApiDoc.Models.School;
using System;

namespace Appeon.ApiDoc.IModelStoreExamples
{
    public class SetPropertyStateExample
    {
        private readonly SchoolContext _context;

        public SetPropertyStateExample(SchoolContext dataContext)
        {
            // Set data context
            _context = dataContext;
        }

        public void Example2()
        {
            IModelStore<Course> courses = 
                new ModelStore<Course>(ChangeTrackingStrategy.PropertyState);

            // 检索出courses中的数据
            courses.Retrieve(_context);

            // 输出Model中第一行记录中CourseID属性的状态
            var state = courses.GetPropertyState(0, "CourseID");
            Console.WriteLine("Before SetPropertyState:");
            Console.WriteLine("Property State = {0}", state.ToString());

            // 修改Model中第一行记录中CourseID属性的状态
            courses.SetPropertyState(0, "CourseID", PropertyState.Modified);

            // 输出修改后Model中第一行记录中CourseID属性的状态
            state = courses.GetPropertyState(0, "CourseID");
            Console.WriteLine("After SetPropertyState:");
            Console.WriteLine("Property State = {0}", state.ToString());

            /* This code example produces the following output:
            
            Before SetPropertyState:
            Property State = NotModified
            After SetPropertyState:
            Property State = Modified
            */
        }
    }
}
