IDataStore.ExportJson(DwBuffer dwbuffer, int startRow, int endRow, short startColumn, short endColumn, MappingMethod mappingMethod) Method

.NET Standard 2.x | Current Version(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

将行（从指定的起始位置到结束位置）和列（从指定的起始位置到结束位置）从指定的缓冲区导出到标准格式JSON字符串。

Exports the rows (from the specified starting position to the ending position) and the columns (from the specified starting position to the ending position) from the specified buffer to a standard-format JSON string.

Namespace: Appeon.Data.PowerBuilder

Assembly: Appeon.Data.PowerBuilder.dll

Syntax

  public string ExportJson(DwBuffer dwbuffer, int startRow, int endRow, short startColumn, short endColumn, MappingMethod mappingMethod);

Parameters

dwbuffer Appeon.Data.PowerBuilder.DwBuffer

A value of the DwBuffer enumerated datatype identifying the DataWindow buffer from which you want to export the data.

startRow System.Int32

The zero-based number of the first detail row in the buffer that you want to export.

endRow System.Int32

The zero-based number of the last detail row in the buffer that you want to export.

If it is negative, it indicates the rest of rows.

startColumn System.Int16

The zero-based number of the first column in the buffer that you want to export.

If it is negative, 0 is used.

endColumn System.Int16

The zero-based number of the last column in the buffer that you want to export.

If it is negative, it indicates the rest of columns.

mappingMethod Appeon.Data.MappingMethod

A value of the MappingMethod enumerated datatype specifying the method for mapping columns.

Returns

System.String

Returns the JSON string if it succeeds.

Remarks

See JSON formats for details about the standard-format JSON.

Examples

The following code example demonstrates export department records to JSON string using startRow,endRow,startColumn and endColumn parameters.

using Appeon.Data;
using Appeon.Data.PowerBuilder;
using System;

namespace Appeon.ApiDoc.IDataStoreExamples
{
    public class ExportJsonExample
    {
        private SchoolContext _context;

        public ExportJsonExample(SchoolContext dataContext)
        {
            // Set Data Context
            _context = dataContext;
        }

        public void Example9()
        {
            // Instantiate the datastore with datawindow:d_department
            var datastore = new DataStore("d_department", _context);

            datastore.Retrieve();

            // 导出primary buffer内从第1行到第11行，从第2列到第3列的数据
            string json = datastore.ExportJson(
                DwBuffer.Primary, 0, 10, 1, 2, MappingMethod.JsonIndex);

            Console.WriteLine(json);

            /*This export json file is :

            ExportJson9.json  
            */
        }
    }
}
