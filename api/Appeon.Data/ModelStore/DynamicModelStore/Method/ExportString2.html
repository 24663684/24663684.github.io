IModelStore<TModel>.ExportString(int startRow, int endRow, bool colheading = true) Method

.NET Standard 2.x | Current Version(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

将ModelStore 中的model objects的数据（从指定的起始索引到结束索引）导出为字符串数据，默认导出TModel的属性名称。

Exports data of the model objects (from the specified starting index to the ending index) from the ModelStore to a string. The property names of TModel are exported by default.

Namespace: Appeon.Data

Assembly: Appeon.Data.dll

Syntax

  public string ExportString(int startRow, int endRow, bool colheading = true);

Parameters

startRow System.Int32

The zero-based index for the starting position in ModelStore.

endRow System.Int32

The zero-based index for the ending position in ModelStore.

If it is negative, it indicates the rest of model objects.

colheading System.Boolean

True: The names of properties in TModel are exported.

False: The names of properties TModel are not exported.

Default is true.

Returns

System.String

Returns the string if it succeeds.

Remarks

Values in are separated by tabs. A newline character (For non-Unix platforms, a string containing "\r\n" is a string containing "\n" for Unix platforms) is placed at the end of each line.
