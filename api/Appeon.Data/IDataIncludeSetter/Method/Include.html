IDataIncludeSetter.Include<TKey>(string name, Func<TModel, TKey> property) Method

.NET Standard 2.x | Current Version (0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha (current)

将TModel对象的一个embedded property的数据添加到IDataPacker对象, 并指定该数据的Name。

Adds data of an embedded property of TModel object to the IDataPacker object, and specifies a name for the data.

Namespace: Appeon.Data

Assembly: Appeon.Data.dll

Syntax

IDataIncludeSetter<TModel> Include<TKey>(string name, Func<TModel, TKey> property);

Type Parameter

TKey

The type of the embedded property in the TModel object to be included.

Parameters

name System.String

The name specified for the data.

property System.Func<in T, out TResult>>

A function used to specify the embedded property in the TModel object to be included.

Returns

Appeon.Data.IDataIncludeSetter

Returns the current IDataIncludeSetter object.

Remarks

If you use IDataPacker.AddModel to add the data of a TModel object to the IDataPacker object and name it Key1, then use the IDataIncludeSetter.Include method to add data of an embedded property of the TModel object to the IDataPacker object and name it Key2. Then actually name will be combined to Key1.Key2.

If you use IDataPacker.AddModels to add the data of a collection of TModel objects to the IDataPacker object and name it Key1, then use the IDataIncludeSetter.Include method to add data of an embedded property of the TModel object to the IDataPacker object and name it Key2. Then actually name will be combined to Key1[n].Key2, n is the zero-based index of a TModel object in its collection.

Examples

以下案例将TModel对象及嵌套数据打包

using Appeon.ApiDoc.Models.School;
using Appeon.Data;
using System;
using System.Collections.Generic;

namespace Appeon.ApiDoc.IDataIncludeSetterExamples
{
    public class IncludeExample
    {
        private readonly SchoolContext _context;

        public IncludeExample(SchoolContext dataContext)
        {
            // Sets the data context.
            _context = dataContext;
        }

        public void Example()
        {
            var packer = new DataPacker();

            var modelmapping = new ModelMapper(_context);

            var items = modelmapping.Load<CourseInfo>(2).IncludeAll().ToList();

            // 将items对象及嵌套数据添加到packer对象中
            string key = "Models";
            packer.AddModels(key, items).Include("Students", m => m.Students);

            // 计算packer中数据总数，并输出
            Console.WriteLine("The count is: {0}",
                packer.Count.ToString());

            // 获取packer中所有key值，并输出
            string keys = "";
            for (int i = 0; i < packer.Count; i++)
            {
                keys = keys + packer.Keys[i] + ",";
            }
            Console.WriteLine("The key values is: {0}", keys);

            /* This code example produces the following output:
            
            The count is: 4
            The key values is: Models,Models[0].Students,Models[1].Students,
            Models[2].Students,
            */
        }
    }
}
