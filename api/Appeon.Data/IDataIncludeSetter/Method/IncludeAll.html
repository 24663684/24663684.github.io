IDataIncludeSetter.IncludeAll() Method

.NET Standard 2.x | Current Version (0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha (current)

将TModel对象的所有embedded properties的数据添加到IDataPacker对象。

Adds data of all embedded properties of TModel object to the IDataPacker object.

Namespace: Appeon.Data

Assembly: Appeon.Data.dll

Syntax

void IncludeAll();

Remarks

If you use IDataPacker.AddModel to add the data of a TModel object to the IDataPacker object and name it Key1, then use the IDataIncludeSetter.IncludeAll method to add data of all embedded properties of the TModel object to the IDataPacker object. Then actually name will be combined to Key1.PropertyName for the data from each property, PropertyName is the name of the property.

If you use IDataPacker.AddModels to add the data of a collection of TModel objects to the IDataPacker object and name it Key1, then use the IDataIncludeSetter.IncludeAll method to add data of all embedded properties of the TModel object to the IDataPacker object. Then actually name will be combined to Key1[n].PropertyName for the data from each property, n is the zero-based index of a TModel object in its collection, PropertyName is the name of the property.

Examples

以下案例将TModel对象及所有嵌套数据打包

using Appeon.ApiDoc.Models.School;
using Appeon.Data;
using System;
using System.Collections.Generic;

namespace Appeon.ApiDoc.IDataIncludeSetterExamples
{
    public class IncludeAllExample
    {
        private readonly SchoolContext _context;

        public IncludeAllExample(SchoolContext dataContext)
        {
            // Sets the data context.
            _context = dataContext;
        }

        public void Example()
        {
            var packer = new DataPacker();

            var modelmapping = new ModelMapper(_context);

            var items = modelmapping.Load<CourseInfo>(2).IncludeAll().ToList();

            // 将items对象及所有嵌套数据添加到packer对象中
            string key = "Models";
            packer.AddModels(key, items)
                  .IncludeAll();

            // 计算packer中数据总数，并输出
            Console.WriteLine("The count is: {0}",
                packer.Count.ToString());

            // 获取packer中所有key值，并输出
            string keys = "";
            for (int i = 0; i < packer.Count; i++)
            {
                keys = keys + packer.Keys[i] + ",";
            }
            Console.WriteLine("The key values is: {0}", keys);

            /* This code example produces the following output:
            
            The count is: 10
            The key values is: Models,Models[0].Students,Models[1].Students,
            Models[2].Students,Models[0].Instructors,Models[1].Instructors,
            Models[2].Instructors,Models[0].OnlineFlag,Models[1].OnlineFlag,
            Models[2].OnlineFlag,
            */
        }
    }
}
