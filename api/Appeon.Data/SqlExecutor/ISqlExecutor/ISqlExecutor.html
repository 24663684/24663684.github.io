The `ISqlExecutor` interface represents a `SqlExecutor` object. SQLExecutor是基于ADO.NET进行封装的一个数据库操作接口，提供了更多常用数据库操作方法和参数兼容性，支持直接执行动态SQL语句、SQLBuilder对象、Model自动化查询等功能特性。

The ISqlExecutor interface represents a SqlExecutor object. SQLExecutor is a set of database manipulation interfaces that have encapsulated the core components of ADO.NET. It is a C# replacement for PowerBuilder's embedded SQL feature. Using SQLExecutor you can directly run dynamic SQL statements, perform SQLBuilder object functions, and do model queries.
SQLExecutor supports raw SQL as well as automatic SQL generation, supports lazy loading of data to prevent unnecessary performance bottlenecks, and can bind result-sets to POCO or dynamic models.
