ISqlExecutor.SelectOne<TModel>(string sqlText, params object[] parameters) 方法

.NET Standard 2.x | 当前版本(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

Executes the SQL Select statement and returns the first row in the result returned by the query. All other rows are ignored.

命名空间： Appeon.Data

程序集： Appeon.Data.dll

语法

  public TModel SelectOne<TModel>(string sqlText, params object[] parameters);

Type Parameters

TModel

The datatype of the model object which matches each row in the result set.

You can also specify DynamicModel to TModel if you don't want to define a custom model class.

Parameters

sqlText System.String

A raw SQL Statement which can contain parameter placeholders.

Proper syntax of a parameter placeholder is to use an '@' or ':' symbol prefix on the parameter name (e.g.: @newFirstName), to make a parameterized SQL ( e.g: update Person set FirstName = @newFirstName where PersonId = 1), these placeholders are filled in with actual parameter values when the SQL Statement executes.

parameters System.Object[]

(Optional) One or more ParamValue objects which containing the values corresponding to the SQL parameter identifier names;

You can also pass the arguments directly in the order in which they appear in sqlText without using the ParamValue object.

Returns

TModel

Returns a TModel object which represents the first row of of the result set. Returns null if there are no rows in the result set.

备注

You can use the format @parametername as a parameter placeholder in the raw SQL Statement.

Examples

执行SQL语句，将查询出的数据放入Model中.只会返回一条查询结果

using Appeon.Data;
using System;

namespace Appeon.ApiDoc.ISqlExecutorExamples
{
    public class SelectOneExample
    {
        private readonly SchoolContext _context;

        public SelectOneExample(SchoolContext dataContext)
        {
            // Set data context
            _context = dataContext;
        }

        public void Example1()
        {
            // 定义SQL语句
            var sql = "select CourseID, Title from Course" +
                      " where CourseID = 8001";

            // 执行SQL, 并将查询出的数据放入Model中.
            var master = _context.SqlExecutor.SelectOne<DynamicModel>(sql);

            Console.WriteLine("Title = {0}.",
                              master.GetValue(1));

            /*This code produces the following output:

            Title = Test Course. 
           */

        }
    }
}
