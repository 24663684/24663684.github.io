ISqlExecutor.SelectLazy<TModel>(ISqlQueryBuilder queryBuilder, params object[] parameters) 方法

执行由ISqlQueryBuilder对象生成的SQL Select语句并返回一个IEnumerable <TModel>对象，该对象可用于延迟加载结果集。

Executes the SQL Select statement generated by an ISqlQueryBuilder object and returns an IEnumerable <TModel> object that can be used to load the result set lazily.

命名空间： Appeon.Data

程序集： Appeon.Data.dll

语法

  public IEnumerable<TModel> SelectLazy<TModel>(ISqlQueryBuilder queryBuilder, params object[] parameters);

Type Parameters

TModel

The datatype of the model object which matches each row in the result set.

You can also specify DynamicModel to TModel if you don't want to define a custom model class.

Parameters

queryBuilder Appeon.Data.ISqlQueryBuilder

A SqlQueryBuilder object which can be used to generate a SQL Select statement.

parameters System.Object[]

(Optional) One or more ParamValue objects which containing the values corresponding to the SQL parameter identifier names;

You can also pass the arguments directly in the order in which they appear in the SQL Select statement generated by queryBuilder without using the ParamValue object.

Returns

System.Collections.Generic.IEnumerable<TModel>

Returns an IEunmerable<TModel> object which can load the result set lazily.

Examples

执行SQL语句，将查询出的数据放入枚举中.使用sqlbuild语法格式生成查询语句

using Appeon.Data;
using System;

namespace Appeon.ApiDoc.ISqlExecutorExamples
{
    public class SelectLazyExample
    {
        private readonly SchoolContext _context;

        public SelectLazyExample(SchoolContext dataContext)
        {
            // Set data context
            _context = dataContext;
        }

        public void Example2()
        {
            // 声明SqlQueryBuilder
            ISqlQueryBuilder sql = new SqlQueryBuilder();

            // 定义查询参数和查询列
            int param;
            string[] columns = { "CourseID", "Title" };

            // 定义SQL语句
            sql.Select(columns)
               .From("Course")
               .Where("CourseID", SqlBuilder.Parameter<int>("id"));

            // open当前connection
            _context.CurrentConnection.Open();

            // 执行SQL, 并将查询出的数据放入Model中.
            param = 8001;
            var master = _context.SqlExecutor.SelectLazy<DynamicModel>(sql, param);

            // close当前connection
            _context.CurrentConnection.Close();

            // 循环输出查询出的数据
            foreach (var model in master)
            {
                Console.WriteLine("Title = {0}.",
                              model.GetValue(1));
            }

            /*This code produces the following output:

            Title = Test Course. 
           */            

        }
    }
}
