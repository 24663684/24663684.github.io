ISqlExecutor.Scalar<TValue>(ISqlQueryBuilder sqlQueryBuilder, params object[] parameters) 方法

.NET Standard 2.x | 当前版本(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

Executes the SQL Select statement created by an ISqlQueryBuilder object and returns the first column of the first row in the result returned by the query. All other columns and rows are ignored.

命名空间： Appeon.Data

程序集： Appeon.Data.dll

语法

  public TValue Scalar<TValue>(ISqlQueryBuilder sqlQueryBuilder, params object[] parameters);

Type Parameters

TValue

The datatype of the first column in the result set.

Parameters

sqlQueryBuilder Appeon.Data.ISqlQueryBuilder

A SqlQueryBuilder object which can be used to generate a SQL Select statement.

parameters System.Object[]

(Optional) One or more ParamValue objects which containing the values corresponding to the SQL parameter identifier names;

You can also pass the arguments directly in the order in which they appear in the SQL Select statement generated by sqlQueryBuilder without using the ParamValue object.

Returns

TValue

The first column of the first row in the result set.

If the first column of the first row in the result set is not found, a null reference is returned.

Examples

执行SQL语句，主要用于查询数据库中的字段.使用sqlbuild语法格式执行

using Appeon.Data;
using System;

namespace Appeon.ApiDoc.ISqlExecutorExamples
{
    public class ScalarExample
    {
        private readonly SchoolContext _context;

        public ScalarExample(SchoolContext dataContext)
        {
            // Set data context
            _context = dataContext;
        }

        public void Example2()
        {
            // 定义SqlQueryBuilder
            var id = 8001;
            ISqlQueryBuilder sql = new SqlQueryBuilder();

            sql.From("Course")
               .Where("CourseID", SqlBuilder.Parameter<int>("id"));
            sql.Select("Title");
            
            // 执行SQL, 并返回查询结果
            var title = _context.SqlExecutor.Scalar<string>(sql, id);

            Console.WriteLine("Title = {0} ", title);

            /*This code produces the following output:

            Title = test course 
           */

        }
    }
}
