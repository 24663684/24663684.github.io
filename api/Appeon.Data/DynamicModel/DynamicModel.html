一个密封类，表示一个不需要指定属性及属性的数据类型的动态model object。

它可用于在数据库中执行SQL查询后接收结果集。

A sealed class which represents a dynamic model object that you don't need to specify properties and properties's datatypes.

It can be used to receive the result set after executing a SQL query in database.
