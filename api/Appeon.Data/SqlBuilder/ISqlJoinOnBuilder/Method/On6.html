ISqlJoinOnBuilder.On(string left, SqlBinaryOperator sqlOperator, ISqlBuilder sqlBuilder) 方法

.NET Standard 2.x | 当前版本(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

创建ON子句, 向ON子句添加搜索条件，在运算符左侧指定SQL表达式，在右侧指定SQL子查询。

Creates the ON clause, and adds a search condition to the ON clause, specifies the SQL expression on the left side of the operator, and specify the SQL subquery on the right side.

命名空间： Appeon.Data

程序集： Appeon.Data.dll

语法

  ISqlJoinOnAndOr On(string left, SqlBinaryOperator sqlOperator, ISqlBuilder sqlBuilder);

Parameters

left System.string

A SQL expression on the left of the operator.

SqlBinaryOperator Appeon.Data.SqlBinaryOperator

A enumeration value of SqlBinaryOperator, it's the operator to test the two expression on the left and right.

sqlBuilder Appeon.Data.ISqlBuilder

A ISqlBuilder object which represents a SQL subquery on the right of the operator.

Returns

Appeon.Data.ISqlJoinOnAndOr

Returns the ISqlJoinOnAndOr object which can be used to add more search conditions for current ON clause.

Examples

一表连接另外一个表时，使用On设置连接条件

using Appeon.Data;
using System;

namespace Appeon.ApiDoc.ISqlJoinOnBuilderExamples
{
    public class OnExample
    {
        private readonly SchoolContext _context;

        public OnExample(SchoolContext dataContext)
        {
            // Set data context
            _context = dataContext;
        }

        public void Example6()
        {
            var sqlQueryBuilder = new SqlQueryBuilder();

            // 设置一个查询，用来获取最大DepartmentID
            var sqlBuilder = new SqlQueryBuilder();
            sqlBuilder.SelectRaw("max(DepartmentID)")
                .From("Department");

            // 获取Department表中DepartmentID，Name; Course表中Title数据，
            // Department和Course表关联条件为Department.DepartmentID 小于以上查询获取的最大DepartmentID
            sqlQueryBuilder
                .Select("Department.DepartmentID")
                .Select("Department.Name")
                .Select("Course.Title")
                .From("Department")
                .Join("Course")
                .On("Department.DepartmentID",
                    SqlBinaryOperator.LessThan, sqlBuilder);

            // 将sqlQueryBuilder语法转换成context对应数据库的可执行语句
            string sql = sqlQueryBuilder.ToSqlString(_context);

            Console.WriteLine(sql);
            
            /*This code example produces the following output:           
              SELECT
                [Department].[DepartmentID],
                [Department].[Name],
                [Course].[Title]
              FROM [Department] JOIN [Course]
              ON [Department].[DepartmentID] < (SELECT
                                                    MAX([DepartmentID])
                                                FROM [Department])
            */
        }
    }
}
