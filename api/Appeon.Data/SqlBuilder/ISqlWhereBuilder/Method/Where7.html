ISqlWhereBuilder.Where(ISqlWhereCondition condition) 方法

.NET Standard 2.x | 当前版本(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

创建WHERE子句, 用一个ISqlWhereCondition对象向WHERE子句添加搜索条件。

Creates the WHERE clause, and adds a search condition to the WHERE clause by a ISqlWhereCondition object.

命名空间： Appeon.Data

程序集： Appeon.Data.dll

语法

  ISqlWhereAndOr Where(ISqlWhereCondition condition);

Parameters

condition Appeon.Data.ISqlWhereCondition

A ISqlWhereCondition which represents a search condition for WHERE Clause.

This object can be created by using SqlBuilder.Where method and so on.

Returns

Appeon.Data.ISqlWhereAndOr

Returns the ISqlWhereAndOr object which can be used to add more search conditions for current WHERE clause.

备注

Examples

使用自定义的ISqlWhereCondition对象

using Appeon.Data;
using System;

namespace Appeon.ApiDoc.ISqlWhereBuilderExamples
{
    public class WhereExample
    {
        private readonly SchoolContext _context;

        public WhereExample(SchoolContext dataContext)
        {
            // Set data context
            _context = dataContext;
        }

        public void Example7()
        {
            // 声明SqlQueryBuilder
            ISqlQueryBuilder sqlQueryBuilder = new SqlQueryBuilder();

            // 定义Where查询
            ISqlWhereCondition where =
                SqlBuilder.Where("DepartmentID", SqlBuilder.Parameter<int>("id"));

            // 定义查询参数和查询列
            int deptid = 1;
            string[] columns = { "CourseID", "Title" };

            // 定义SQL语句， 并加入自定义的Where查询
            sqlQueryBuilder.Select(columns)
                            .From("Course", "a")
                            .Where(where);

            // Show SQL
            Console.WriteLine("The SQL statement: \n{0}",
                             sqlQueryBuilder.ToSqlString(_context));

            // Execute SQL
            var courses = _context.SqlExecutor
                .Select<DynamicModel>(sqlQueryBuilder, deptid);

            Console.WriteLine();
            Console.WriteLine("Title (first row) : {0}.",
                              courses[0].GetValue("Title"));

            /*This code produces the following output:

            The SQL statement:
            SELECT
             [CourseID],
            [Title]
            FROM [Course] AS [a]
            WHERE ([DepartmentID] = @id)

            Title (first row) : Chemistry.
           */
        }
    }
}
