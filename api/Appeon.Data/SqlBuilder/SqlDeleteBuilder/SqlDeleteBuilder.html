代表一个用于构建SQL DELETE statement的SQL Builder object.

The SQL DELETE statement可用于从数据库移除一行或者多行。

Represents a SQL Builder object used to build a SQL DELETE statement.

The SQL DELETE statements can be used to remove one or more rows from the database.
