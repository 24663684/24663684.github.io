ISqlWhereAndOr<TBuilder>.AndWhere(string left, ISqlBuilder sqlBuilder) 方法

.NET Standard 2.x | 当前版本(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

向WHERE子句添加AND逻辑运算符和一个搜索条件，在运算符左侧指定SQL表达式，在右侧指定SQL子查询。运算符使用'='。

Adds the AND logical operator and a search condition to the WHERE clause, specifies the SQL expression on the left side of the operator, and specify the SQL subquery on the right side. The operator is '='.

命名空间： Appeon.Data

程序集： Appeon.Data.dll

语法

  TBuilder AndWhere(string left, ISqlBuilder sqlBuilder);

Parameters

left System.String

A SQL expression on the left of the operator.

sqlBuilder Appeon.Data.ISqlBuilder

A ISqlBuilder object which represents a SQL subquery on the right of the operator.

Returns

TBuilder

Returns the TBuilder object which can be used to add more search conditions for current WHERE clause.

备注

Examples

以下案例查询Course表中列Title的数据，并且满足a.CourseID>@courid and a.DepartmentID=子查询结果的条件

using Appeon.Data;
using System;

namespace Appeon.ApiDoc.ISqlWhereAndOrExamples
{
    public class AndWhereExample
    {
        private readonly SchoolContext _context;

        public AndWhereExample(SchoolContext dataContext)
        {
            // Set data context
            _context = dataContext;
        }

        public void Example5()
        {
            // 声明SqlQueryBuilder
            var sqlbuilder = new SqlQueryBuilder();

            // 定义子查询
            var deptsql = new SqlQueryBuilder();
            deptsql.Select("DepartmentID")
                .From("Department")
                .Where("Name", SqlBuilder.Parameter<string>("id"));

            // 定义SQL语句， 并加入自定义的子查询
            sqlbuilder.Select("Title")
                .From("Course", "a")
                .Where("CourseID", SqlBinaryOperator.GreaterThan,
                      SqlBuilder.Parameter<int>("courid"))
               .AndWhere("a.DepartmentID", deptsql);

            string sql = sqlbuilder.ToSqlString(_context);

            Console.WriteLine(sql);

            /*This code produces the following output:

            SELECT
            [Title]
            FROM [Course] AS [a]
            WHERE ([CourseID] > @courid
            AND [a].[DepartmentID] = (SELECT
            [DepartmentID]
            FROM [Department]
            WHERE ([Name] = @id)))
           */

        }
    }
}
