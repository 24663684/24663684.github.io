ISqlHavingAndOr<TBuilder>.OrHaving(string left, SqlBinaryOperator sqlOperator, ISqlParameter parameter) 方法

.NET Standard 2.x | 当前版本(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

向HAVING子句添加OR逻辑运算符和一个搜索条件，在运算符左侧指定SQL表达式, 在运算符右侧指定一个SQL参数。

Adds the OR logical operator and a search condition to the HAVING clause, specifies an SQL expression to the left of the operator and an SQL parameter to the right of the operator.

命名空间： Appeon.Data

程序集： Appeon.Data.dll

语法

  TBuilder OrHaving(string left, SqlBinaryOperator sqlOperator, ISqlParameter parameter);

Parameters

left System.String

A SQL expression on the left of the operator.

SqlBinaryOperator Appeon.Data.SqlBinaryOperator

A enumeration value of SqlBinaryOperator, it's the operator to test the two expression on the left and right.

parameter Appeon.Data.ISqlParameter

A ISqlParameter object which represents a SQL parameter.

It can be created by calling SqlBuilder.Parameter method.

Returns

TBuilder

Returns the TBuilder object which can be used to add more search conditions for current HAVING clause.

备注

Examples

增加分组条件

using Appeon.Data;
using System;

namespace Appeon.ApiDoc.ISqlHavingAndOrExamples
{
    public class OrHavingExample
    {
        private readonly SchoolContext _context;

        public OrHavingExample(SchoolContext dataContext)
        {
            // Set data context
            _context = dataContext;
        }

        public void Example2()
        {
            var sqlQueryBuilder = new SqlQueryBuilder();

            // 在StudentGrade表中根据StudentID和CourseID分组，获取每个分组的Grade总和，
            // 分组条件为StudentID = 6，或者StudentID等于一个输入参数
            // 输入参数在执行sql时指定
            sqlQueryBuilder
                .Select("StudentID")
                .Select("sum(Grade)")
                .From("StudentGrade")
                .GroupBy("StudentID, CourseID")
                .HavingRaw("StudentID = 6")
                .OrHaving("StudentID",
                           SqlBinaryOperator.Equals,
                           SqlBuilder.Parameter<int>("studentid"));

            // 将sqlQueryBuilder语法转换成context对应数据库的可执行语句
            string sql = sqlQueryBuilder.ToSqlString(_context);

            // execute sql
            // 在执行sql时需传入参数
            var studentGrade = _context.SqlExecutor.Select<DynamicModel>(sql, 8);

            Console.WriteLine(sql);

            /*This code example produces the following output:           
              SELECT
                [StudentID],
                sum(Grade)
              FROM [StudentGrade]
              GROUP BY
                [StudentID],
                [CourseID]
                HAVING ([StudentID] = 6
                OR [StudentID] = @studentid)
           */
        }
    }
}
