ISqlUpdateBuilder.Set(string columnName, ISqlBuilder queryBuilder) 方法

.NET Standard 2.x | 当前版本(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

添加要更新的列，指定要更新到该列的SQL语句(由一个ISqlBuilder 对象指定).

Adds a column to be updated, specifying a SQL subquery (specified by an ISqlBuilder object) to be updated to the column.

命名空间： Appeon.Data

程序集： Appeon.Data.dll

语法

  ISqlUpdateBuilder Set(string columnName, ISqlBuilder queryBuilder);

Parameters

columnName System.String

The name of the column to be updated.

queryBuilder Appeon.Data.ISqlBuilder

A ISqlBuilder object which represents a SQL subquery.

Returns

Appeon.Data.ISqlUpdateBuilder

Returns the current ISqlUpdateBuilder object.

备注

Examples

演示Set(string columnName, ISqlBuilder queryBuilder)方法。以下案例更新OnlineCourse表courseid=2030的记录使用子查询获得的数据值来修改url的值

using Appeon.Data;
using System;

namespace Appeon.ApiDoc.ISqlUpdateBuilderExamples
{
    public class SetExample
    {
        private SchoolContext _context;

        public SetExample(SchoolContext dataContext)
        {
            // Set Data Context
            _context = dataContext;
        }

        public void Example3()
        {
            // 声明SqlQueryBuilder
            var sqlbuilder = new SqlUpdateBuilder();
            var subQuery = new SqlQueryBuilder();

            // 得到子查询语句
            subQuery.Select("url")
                .From("OnlineCourse")
                .Where("courseid","2021");

            // 更新OnlineCourse表courseid=2030的记录
            // 使用子查询获得的数据值修改url列的值
            sqlbuilder.Update("OnlineCourse")
                   .Set("url", subQuery)
                   .Where("courseid", "2030");

            // 将sqlbuilder语法转换成context对应数据库的可执行语句
            string sql = sqlbuilder.ToSqlString(_context);

            Console.WriteLine(sql);
            
            /*This code produces the following output:
            
            UPDATE [OnlineCourse] SET [url] = (SELECT
            [url]
            FROM [OnlineCourse]
            WHERE ([courseid] = 2021))
            WHERE ([courseid] = 2030)
            */
        }
    }
}
