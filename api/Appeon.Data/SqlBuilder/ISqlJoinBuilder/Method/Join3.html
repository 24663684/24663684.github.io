ISqlJoinBuilder.Join(string table, string alias, string schema) 方法

.NET Standard 2.x | 当前版本(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

不指定联接运算符的类型, 将原始表源与新表源联接以创建一个联接表。 需要为新表源指定架构和别名。

Joins the original table source with the new table source, without specifying the type of the join operator, to create a joined table. You need to specify a schema and an alias for the new table source.

命名空间： Appeon.Data

程序集： Appeon.Data.dll

语法

  ISqlJoinOnBuilder Join(string table, string alias, string schema);

Parameters

table System.String

The new table source.

alias System.String

The alias of the new table source.

schema System.String

The schema of the new table source.

Returns

Appeon.Data.ISqlJoinOnBuilder

Returns a ISqlJoinOnBuilder object which can be used to create the condition on which the join is based.

Examples

使用join将一个表与另外一表关联

using Appeon.Data;
using System;

namespace Appeon.ApiDoc.ISqlJoinBuilderExamples
{
    public class JoinExample
    {
        private readonly SchoolContext _context;

        public JoinExample(SchoolContext dataContext)
        {
            // Set data context
            _context = dataContext;
        }

        public void Example3()
        {
            var sqlQueryBuilder = new SqlQueryBuilder();

            // 获取Department表中DepartmentID，Name;Course表中Title数据，
            // Department和Course表关联条件为Department.DepartmentID = Course.DepartmentID
            // 关联表Course的别名指定为CourseTable,指定Course表schema为dbo
            sqlQueryBuilder
                .Select("Department.DepartmentID")
                .Select("Department.Name")
                .Select("CourseTable.Title")
                .From("Department")
                .Join("Course", "CourseTable", "dbo")
                .OnRaw("Department.DepartmentID = CourseTable.DepartmentID");

            // 将sqlQueryBuilder语法转换成context对应数据库的可执行语句
            string sql = sqlQueryBuilder.ToSqlString(_context);

            Console.WriteLine(sql);
            
            /*This code example produces the following output:           
              SELECT
                [Department].[DepartmentID],
                [Department].[Name],
                [CourseTable].[Title]
              FROM [Department]
              JOIN [dbo].[Course] AS [CourseTable]
              ON [Department].[DepartmentID] = [CourseTable].[DepartmentID]
            */
        }
    }
}
