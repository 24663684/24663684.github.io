SqlBuilder.WhereIsNotNull(string left) 方法

创建一个ISqlWhereCondition对象, 表示在构造WHERE子句时可以使用的搜索条件，使用IS NOT NULL运算符搜索非null值。

Creates an ISqlWhereCondition object which represents a search condition can be used when building the WHERE clause, uses IS NOT NULL operator to search for values that are not null.

命名空间： Appeon.Data

程序集： Appeon.Data.dll

语法

 public static ISqlWhereCondition WhereIsNotNull(string left);

Parameters

left System.String

The SQL expression on the left of the IS NOT NULL operator.

Returns

Appeon.Data.ISqlWhereCondition

Returns the ISqlWhereCondition object which can be used to add more search conditions for the WHERE clause.

备注

Examples

判断当前表中指定的字段是否为空，将非空的数据查询出来返回Model中

using Appeon.Data;
using System;

namespace Appeon.ApiDoc.SqlBuilderExamples
{
    public class WhereIsNotNullExample
    {
        private SchoolContext _context;

        public WhereIsNotNullExample(SchoolContext dataContext)
        {
            // Set Data Context
            _context = dataContext;
        }

        public void Example()
        {
            var sqlquerybuilder = new SqlQueryBuilder();
            
            // 定义Where查询
            var where = SqlBuilder.WhereIsNotNull("Title");

            // 创建sql语法
            var query = sqlquerybuilder
                .Select("*")
                .From("Course")
                .Where(where);

            // 转换成context对应数据库的可执行语句
            string sql = query.ToSqlString(_context);

            Console.WriteLine(sql);
            
            /*This code example produces the following output:   
             
            SELECT
            *
            FROM [Course]
            WHERE ([Title] IS NOT NULL)
            */
        }
    }
}
