SqlBuilder.WhereExists(ISqlBuilder sqlBuilder) 方法

.NET Standard 2.x | 当前版本(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

创建一个ISqlWhereCondition对象, 表示在构造WHERE子句时可以使用的搜索条件，使用EXISTS带一个子查询的来测试子查询返回的行是否存在。

Creates an ISqlWhereCondition object which represents a search condition can be used when building the WHERE clause, uses EXISTS with a subquery to test for the existence of rows returned by the subquery.

命名空间： Appeon.Data

程序集： Appeon.Data.dll

语法

 public static ISqlWhereCondition WhereExists(ISqlBuilder sqlBuilder);

Parameters

sqlBuilder Appeon.Data.ISqlBuilder

A ISqlBuilder object which represents a SQL subquery on the right of the EXISTS operator.

Returns

Appeon.Data.ISqlWhereCondition

Returns the ISqlWhereCondition object which can be used to add more search conditions for the WHERE clause.

备注

Examples

判断当前表中数据是否存在子查询返回的数据中，并将查询结果返回Model中

using Appeon.Data;
using System;

namespace Appeon.ApiDoc.SqlBuilderExamples
{
    public class WhereExistsExample
    {
        private SchoolContext _context;

        public WhereExistsExample(SchoolContext dataContext)
        {
            // Set Data Context
            _context = dataContext;
        }

        public void Example()
        {
            var sqlquerybuilder = new SqlQueryBuilder();

            // 定义子查询
            var subsqlbuilder = new SqlQueryBuilder();
            subsqlbuilder.Select("1")
                .From("Department")
                .Where("departmentid", "c.departmentid");

            // 定义Where查询
            var where = SqlBuilder.WhereExists(subsqlbuilder);

            // 创建sql语法
            var query = sqlquerybuilder
                .Select("*")
                .From("Course", "c")
                .Where(where);

            // 转换成context对应数据库的可执行语句
            string sql = query.ToSqlString(_context);

            Console.WriteLine(sql);
            
            /*This code example produces the following output:   
             
            SELECT
            *
            FROM [Course] AS [c]
            WHERE ( EXISTS (SELECT
            1
            FROM [Department]
            WHERE ([departmentid] = [c].[departmentid])))
            */
        }
    }
}
