SqlBuilder.EmbeddedOn(string left, ISqlParameter parameter) 方法

.NET Standard 2.x | 当前版本(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

创建一个ISqlJoinCondition对象, 表示在构造ON子句时可以使用的(括在括号中的)搜索条件，在运算符左侧指定SQL表达式, 在运算符右侧指定一个SQL参数。操作符使用'='。

Creates an ISqlJoinCondition object which represents a search condition (enclosed in parentheses) can be used when building the ON clause, specifies an SQL expression to the left of the operator and an SQL parameter to the right of the operator. The operator is '='.

命名空间： Appeon.Data

程序集： Appeon.Data.dll

语法

 public static ISqlJoinCondition EmbeddedOn(string left, ISqlParameter parameter);

Parameters

left System.String

A SQL expression on the left of the operator.

parameter Appeon.Data.ISqlParameter

A ISqlParameter object which represents a SQL parameter.

It can be created by calling SqlBuilder.Parameter method.

Returns

Appeon.Data.ISqlJoinCondition

Returns the ISqlJoinCondition object which can be used to add more search conditions for the ON clause.

备注

The new created search condition is enclosed in parentheses. If uses the returned ISqlJoinCondition object to add another search condition, both of these search conditions will be enclosed in the same pair of parentheses.

Examples

演示EmbeddedOn(string left, ISqlParam parameter)以下案例演示Department表join Course表.使用嵌入式条件语句作为连接2个表的关联条件

using Appeon.Data;
using System;

namespace Appeon.ApiDoc.SqlBuilderExamples
{
    public class EmbeddedOnExample
    {
        private SchoolContext _context;

        public EmbeddedOnExample(SchoolContext dataContext)
        {
            // Set Data Context
            _context = dataContext;
        }

        public void Example3()
        {
            ISqlQueryBuilder sqlQueryBuilder = new SqlQueryBuilder();

            // 定义嵌入式的Join条件
            var join = SqlBuilder.EmbeddedOn("Department.DepartmentID",
                    SqlBuilder.Parameter<int>("departmentid"));

            // 创建sql语法
            var query = sqlQueryBuilder
                .Select("Department.DepartmentID")
                .Select("Department.Name")
                .Select("Course.Title")
                .From("Department")
                .Join("Course")
                .On(join);

            // 转换成context对应数据库的可执行语句
            string sql = query.ToSqlString(_context);

            Console.WriteLine(sql);
            
            /*This code produces the following output:
            
            SELECT
            [Department].[DepartmentID],
            [Department].[Name],
            [Course].[Title]
            FROM [Department] JOIN [Course]
            ON [Department].[DepartmentID] = @departmentid
            */
        }
    }
}
