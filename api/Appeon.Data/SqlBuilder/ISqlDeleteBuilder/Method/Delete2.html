ISqlDeleteBuilder.Delete(string table, string schema) 方法

创建SQL DELETE语句并指定要从中删除行的表或视图的名称以及schema。

Creates the SQL DELETE statement and specifies the name and the schema of the table or view from which the rows are to be removed.

命名空间： Appeon.Data

程序集： Appeon.Data.dll

语法

  ISqlDeleteBuilder Delete(string table, string schema);

Parameters

table System.String

The name of the table or view from which the rows are to be removed.

schema System.String

The schema of the table or view from which the rows are to be removed.

Returns

Appeon.Data.ISqlDeleteBuilder

Returns the current ISqlDeleteBuilder object.

备注

Examples

演示ISqlDeleteBuilder Delete(string table, string schema)方法。构造deletebuilder语句删除Department表中name="English"的记录，并指定Department表schema

using Appeon.Data;
using System;

namespace Appeon.ApiDoc.ISqlDeleteBuilderExamples
{
    public class DeleteExample
    {
        private SchoolContext _context;

        public DeleteExample(SchoolContext dataContext)
        {
            // Set Data Context
            _context = dataContext;
        }

        public void Example2()
        {
            // 声明SqlDeleteBuilder
            var sqlbuilder = new SqlDeleteBuilder();

            // 构建Delete语句，删除Department表中name="English"的记录
            sqlbuilder.Delete("Department", "dbo")
                .WhereValue("name", SqlBinaryOperator.Equals, "English");

            // 将sqlbuilder语法转换成context对应数据库的可执行语句
            string sql = sqlbuilder.ToSqlString(_context);

            Console.WriteLine(sql);

            /*This code produces the following output:
            
            DELETE
            FROM [dbo].[Department]
            WHERE ([name] = N'English')    
            */
        }
    }
}
