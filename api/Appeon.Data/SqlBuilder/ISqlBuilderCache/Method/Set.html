ISqlBuilderCache.Set(string sqlBuilderName, ISqlBuilder sqlBuilder, bool readOnly = true) 方法

.NET Standard 2.x | 当前版本(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

将ISqlBuilder对象缓存到ISqlBuilderCache对象中，并指定ISqlBuilder对象的缓存名称。 您还可以指定ISqlBuilder对象再从缓存中获取时是否为只读，默认为只读。

Caches an ISqlBuilder object into the ISqlBuilderCache object and specifies a cache name for the ISqlBuilder object. You can also specify whether the ISqlBuilder object is read-only when getting from the cache, the default is read-only.

命名空间： Appeon.Data

程序集： Appeon.Data.dll

语法

void Set(string sqlBuilderName, ISqlBuilder sqlBuilder, bool readOnly = true);

Parameters

sqlBuilderName System.String

The name specified for the ISqlBuilder cached.

sqlBuilder Appeon.Data.ISqlBuilder

The ISqlBuilder object to be cached into the ISqlBuilderCache object.

readOnly System.Boolean

Specify whether the ISqlBuilder object is read-only when getting from the cache.

True, you can only use the Get method to get a ISqlReadOnlyBuilder object from cache.

False, you can use the Get method to get a ISqlReadOnlyBuilder object from cache or use the GetModifiable method to get a TSqlBuilder object.

Default is true.

Remarks

Examples

将ISqlBuilder对象缓存到ISqlBuilderCache中，并为其指定缓存名称

using Appeon.Data;
using System;

namespace Appeon.ApiDoc.ISqlBuilderCacheExamples
{
    public class SetExample
    {
        private readonly SchoolContext _context;

        public SetExample(SchoolContext context)
        {
            _context = context;
        }

        public void Example()
        {
            // 创建一个SqlQueryBuilder对象
            var build = SqlBuilderFactory.Create<ISqlQueryBuilder>("original sql");
            build.Select("select Title from Course wherer CourseID = 1050");

            // 通过SqlBuilderFactory.Cache，生成一个SqlBuilderCache对象
            var cache = SqlBuilderFactory.Cache;

            // 将SqlQueryBuilder注入SqlBuilderCache对象中
            cache.Set("querySql", build, true);

            // 从缓存中获取名为querySql的ISqlReadOnlyBuilder对象
            var get = cache.Get("querySql");

            Console.WriteLine("SqlBuild Name = {0}", get.Name);

            /*This code produces the following output:
            
            SqlBuild Name = original sql
            */
        }
    }
}
