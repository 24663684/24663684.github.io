DataContext.CurrentTransaction 属性

获取一个IAdoDbTransaction对象。 此对象可用于获取有关数据库事务的信息，以及提交或回滚事务。

Gets a IAdoDbTransaction object. This object can be used to get info about the database transaction, as well as commit or rollback the transaction.

命名空间： Appeon.Data

程序集： Appeon.Data.dll

语法

   public virtual IAdoDbTransaction CurrentTransaction { get; }

Property Value

Appeon.Data.IAdoDbTransaction

A IAdoDbTransaction object which can be used to get info about the database transaction, as well as commit or rollback the transaction.

Remarks

Examples

获取指定上下文的当前事务

using System;

namespace Appeon.ApiDoc.DataContextExamples
{
    public class CurrentTransactionExample
    {
        private readonly SchoolContext _context;
        public CurrentTransactionExample(SchoolContext context)
        {
            _context = context;
        }

        public void Example()
        {
            // 开启事务
            _context.BeginTransaction();

            // 获取当前事务
            var conn = _context.CurrentTransaction;

            // 开启事务后，需终止事务
            conn.Rollback();

            // 输出当前事务属性的类型
            Console.WriteLine("Transaction Type = {0}", conn.GetType());

            /* This code example produces the following output:
            
            Transaction Type = Appeon.Data.AdoDbTransaction
            */

        }
    }
}
