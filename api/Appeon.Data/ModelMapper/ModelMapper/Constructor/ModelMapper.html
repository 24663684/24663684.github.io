ModelMapper.ModelMapper(DataContext context) 构造函数

.NET Standard 2.x | 当前版本(0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

使用一个DataContext对象初始化一个ModelMapper类的新实例。

Initializes a new instance of the ModelMapper class by a DataContext object.

命名空间： Appeon.Data

程序集： Appeon.Data.dll

语法

  public ModelMapper(DataContext context);

Parameters

context Appeon.Data.DataContext

A DataContext object，其中包含用于创建数据库会话的上下文信息。

A DataContext object, which includes the context information for creating the database sessions.

备注

除了手动新建ModelMapper实例外，也可以使用实例化的DataContext对象，访问其DataContext.ModelMapper属性获取ModelMapper实例.

Except for manually creating a ModelMapper instance, you can also use an initialized DataContext object to access the DataContext.ModelMapper property and get the ModelMapper instance.

Examples

The following code example demonstrates how to create a ModelMapper object.

using Appeon.Data;
using System;

namespace Appeon.ApiDoc.ModelMapperExamples
{
    public class ModelMapperExample
    {
        private SchoolContext _context;

        public ModelMapperExample(SchoolContext dataContext)
        {
            // Sets the data context.
            _context = dataContext;
        }

        public void Example()
        {
            // Creates the `ModelMapper` object by a `DataContext` object.
            IModelMapper mapper = new ModelMapper(_context);

            Console.WriteLine("{0}.",
                mapper.ToString());

            // In addition to using the constructor, you can also use the 
            // `DataContext.ModelMapper` property to get the `IModelMapper` object.
            var mapper2 = _context.ModelMapper;

            /*This code produces the following output:

            Appeon.Data.ModelMapper.
           */
        }
    }
}
