通过键指定的元素，获取新创建的IDataStore对象, 使用被该元素指定的DataWindow object和一个DataContext对象, 指定用于向IDataStore对象导入数据的映射方法。

Gets a new created IDataStore object by the element specified by the key, using the DataWindow object specified by the element and a DataContext object, specifying the mapping method used for importing data into the IDataStore object.
