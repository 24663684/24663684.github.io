Welcome to PowerBuilder 2019

Welcome to the new Release of PowerBuilder 2019!

With this release we introduce 2 new Non-Visual Business Targets:

  • C# Web API
  • C# Assembly

With these targets you can now migrate your existing business logic to the cloud!

Along with the new targets, PowerBuilder also includes new C# objects for the new targets:

  • [DataStore](./Getting Started - C# DataStore.md)
  • ModelStore
  • ModelMapper
  • SQL Builder
  • SQL Executor
  • OAuth Server

These objects enable you to create awsome .NET Core C# projects easily, fast and with little effort; the PowerBuilder way!

PowerBuilder 2019 now also includes a new C# IDE! This IDE will come included with the Cloud and Universal Editions of PowerBuilder. This IDE is both easy to install and use. It comes with everything you need to start building great PowerBuilder C# projects.

The new Appeon C# IDE features a rich development environment characteristic of a RAD (Rapid Application Development) programming language. It is mostly meant for those who wish to use a different IDE, than PowerBuilder, to develop C# Code. But it comes with pre-built Appeon C# specific tools too. This enables you to develop C# code in a Snap!

We will cover more about the new C# IDE in a separate article. But, for now, let's continue reviewing what else is new in PowerBuilder 2019...

Another big improvement in PowerBuilder is the new PowerBuilder Themes! A feature that has been long awaited for...

With PowerBuilder Themes you can now provide a totally new improved UI experience to your users. The PowerBuilder Themes allows you to change the look and feel of your PowerBuilder objects UI during runtime. It comes included with 3 pre-built themes: Flat Design Blue, Flat Design Gray, and Flat Design Silver.

You can also create your own custom Themes too! And, creating the themes, is as easy as editing a JSON file similar to CSS.
