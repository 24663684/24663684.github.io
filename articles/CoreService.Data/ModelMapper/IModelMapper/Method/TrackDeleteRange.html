IModelMapper.TrackDeleteRange<TModel>(IEnumerable<TModel> models) Method

.NET Standard 2.x | Current version (0.5.0-alpha)

0.5.0-alpha

0.5.1-alpha(current)

追踪一个或者多个数据库表删除操作,将从数据库表删除的数据缓存于TModel中。当IModelMapper.SaveChanges被调用时，一条或者多条SQL DELETE statement 将用缓存在TModel中的数据以及被定义在TModel中的映射信息来产生，并执行。 Tracks one or more database table delete operations. The data that will be deleted from the database table is cached in TModel. When IModelMapper.SaveChanges is called, one or more SQL DELETE statements will be first generated using the data cached in TModel and the mapping information defined in TModel, and then executed.
